Setting Up Workflow

While the standard end users of KRA (Researchers) are able to set their own Action List preferences, there are many more Workflow settings that are configured using the KRA user interface that some Unit and Central Administration users must be familiar with.  On their menu screens, their Workflow menu groups include links for setting up workflow preferences, running reports and setting up rules and workgroups within the Kuali Enterprise Workflow (KEW). Whereas individual users can change their own workflow preferences, the institution’s system administrators or the super users within the functional area are responsible for making changes to the workflow rules.

Options in this menu group are used to set up workflow
