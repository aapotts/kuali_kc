Shared Menu Group

The Shared menu group is a box with a bullet list of links that take you to maintenance document lookup screens for maintenance documents that are shared by more than one document and/or module in the system.  From those lookup screens you are able to edit existing or create new maintenance documents for reference information used throughout KRA.

For basic process information about creating and updating maintenance documents, see Common Maintenance Document Operations in KRA Overview.

The links appearing in the Shared menu group are:

•      Activity Type

•      Approval Type

•      Budget Category

•      Budget Category Mapping

•      Budget Category Maps

•      Budget Category Type

•      Cost Element

•      Custom Attribute
