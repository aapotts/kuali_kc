Credit Type Maintenance Document

     >          >    

Clicking the Credit Type menu link displays the Investigator Credit Type Lookup screen.  Clicking create new displays the Investigator Credit Type screen, which is a maintenance document, but isn’t labeled with the words “Maintenance Document” at the end like others.  Another unique feature is that what would on most maintenance documents be an “Edit…” tab, instead, is labeled a Specify the Credit Type tab that allows you to enter and select the detailed data of the reference table entry.
