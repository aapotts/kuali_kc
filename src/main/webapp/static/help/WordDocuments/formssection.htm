Forms Section

Figure 97 Forms Section

Table 44 Forms Section Definition

Column

Description

Form Name

The government assigned title of a form that may accompany the Grants.Gov opportunity. The official title of the form document

Mandatory

Indicates if a given form is required to submit the proposal to Grants.Gov. Yes no or n/a for not applicable is displayed.  Read-only display.

Include

Indicates if a given form will be included with the proposal submission package. Yes no or n/a for not applicable is displayed.  Read-only display or checkbox.

Desc

Indicates if a given form is available to be included with the proposal submission package. Description of the form’s availability.

Select to Print

Indicates which forms will be printed when the print action is performed. Select the checkbox to include all or clear to include none.  Click within the checkbox  to select the item.  A green check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

print selected forms

All Included - selects all available forms to be printed.  Click to print the forms that are checked in the Select to Print column.  None - Deselects all forms in the print column.
