Kuali Research Administration (KRA) Release Version 1.1 Help

November 1st, 2008

What’s New In This Release?

The changes made in this release were limited to the redesign of the user interface; functionally the product is the same as it was in 1.0.  The user interface was altered in direct response to feedback from multiple design critiques held over the past 9 months.  Participants in the design critiques included all user types; faculty, departmental support staff, and sponsored program office administrators.  Changes to the user interface include:

•      Reorganization of the Proposal and Budget Tabs to take advantage of how typical users currently work

•      Changing tab names and field labels in the Budget area to be more intuitive for a novice or first time user

•      Providing you with a more inclusive Summary view of the budget so that you can quickly ascertain both the overall budget as well as key summary figures

•      Dividing the Personnel and Non-Personnel sections of the budget into two, functionally distinct areas so that you understand exactly where to include key elements in your budget

•      Reorganizing the Personnel tab to be all inclusive so that you do not need to navigate to multiple pages to budget personnel

 

Although the system is sometimes referenced inconsistently online and in print as Kuali Coeus (KC), Kuali COEUS, Kuali Coeus Research Administration (KCRA), or Kuali COEUS Research Administration, this edition of user documentation refers to the application as Kuali Research Administration (KRA) to reflect the logo and screen captures within this software release version as delivered.  Precise clarification of a Board decision to officially add the word Coeus to the name and revision to the logo is pending as of this release.
