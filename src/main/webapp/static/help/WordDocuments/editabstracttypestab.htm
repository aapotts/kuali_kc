Abstract Type

Required.  Enter the alphanumeric code value that is to be used to identify and reference this individual type of abstract.

Description

Required.  Enter a textual name for the abstract type that uniquely represents a summarized explanation of its meaning and/or purpose (for example, a title, several words separated by spaces, or a short phrase).  The abstract, or project summary, is a concise, clear, and brief description of the project. It should outline the problem, the objectives, expected outcomes, including significance of the project to the field being studied. The abstract is usually written in less technical language than the proposal narrative.
