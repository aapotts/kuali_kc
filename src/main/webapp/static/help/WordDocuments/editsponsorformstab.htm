Edit Sponsor Forms Tab

Use the Edit SponsorForms tab to create or maintain a reference table record by specifying the following three required fields of information:

 

Field

Description

Package Name

Required.  Enter the alphanumeric text used to refer to the package (may be a familiar abbreviation).

Package Number

Required.  Enter the numeric value representing the package.

Sponsor Code

Required.  Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions to specify the number associated with a particular sponsor.
