Searching for a Workgroup

The default start mode for searching for workgroups that fit your search criteria is the Default (or blank) mode.  When you select something other than ‘Default’ in the Workgroup Type field, additional search criteria fields appear on the screen, allowing you to search for workgroups of this type by these attributes.

 

Table 93 Search for a WorkGroup Screen Search Criteria Section Field Definition

Field

Description

WorkGroup Name

Optional. Enter a full or partial string of the workgroup name for which you are searching. 

 You do not need to use ‘*’ for a wild card search. Upper and lower case insensitive.

WorkGroup Type

Optional. Leaving the selection blank defaults to the ‘Default’ search mode.

WorkGroup Description

Optional. Enter a full or partial workgroup description. 

 You do not need to use ‘*’ for a wild card search. Upper and lower case insensitive.

WorkGroup Member

Optional. Enter a user ID or select it from the User ID lookup .

WorkGroup ID

Optional. Enter the WorkGroup ID of the workgroup you are searching for. This ID is automatically assigned by workflow when the workgroup is created.

Active Indicator

Optional. Select one of the following:  ‘Active’ – Retrieve all active workgroups;  ‘Inactive’ – Retrieve all inactive workgroups;  ‘’Show All’ – Retrieve both active and inactive workgroups.  The default is ‘Show All.’
