Specify the Credit Type Tab

 

 

Field

Definition

Investigator Credit Type Code

Required.  The value that uniquely identifies this investigator credit type in the system and serves as an abbreviated reference for the actual name appears in the Description field.

Is Active

Check to activate, clear to make inactive.

Description

The text that explains the purpose or meaning of the credit type as it is commonly referred to.

Adds to Hundred

Check to require line item values add up to 100%.
