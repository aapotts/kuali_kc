Edit Person Document Type Tab

The Edit Person Document Type tab requires population of two fields that define an entry in the reference table, one a numeric value, and the other a textual description of the associated value.

 

Field

Definition

Person Document Type Code

Required.  Enter the value that will be used to represent this single piece of information when referenced.

Description

Required.  Enter the familiar name of the person document type you would use to refer to it verbally or in writing.
