Frequency Base Maintenance Document

Use the Frequency Base maintenance document to establish new and to update existing reference table records for Frequency Base Codes.

 

Figure 523 Frequency Base Maintenance Document Screen

 

When your institution enters award information on the KC Award e-doc, you need to use Frequency Base Codes.

Use Frequency Base Codes when entering award information on the Payment, Reports & Terms tab on the KC Award e-doc.  Select Frequency Base Codes when completing the Payment & Invoice Requirements section in the Payment & Invoices panel.

 

Figure 524 KC Award Screen
