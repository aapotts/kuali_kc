Edit Sponsor Template Comments Panel

The Edit Sponsor Template Comments panel lets you create one or more New Sponsor Template Comments.  These comments might include information about the interaction with a sponsor in providing assistance in managing the award.  When synchronized with an award, the Sponsor Template updates the award with the comment information providing consistency for awards with the same Sponsor Template.

The Edit Sponsor Template Comments panel lets you select a Comment Type and a Checklist Print Flag.  It also provides a free-form text box for your comments.
