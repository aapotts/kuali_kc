Document Header – Proposal Log

The Document Header is common to all e-docs.  It displays information about the e-doc.  The fields included in the document header are dependent on the document type:

 

Figure 274 Proposal Log Document Header

 

Table 97 Proposal Log Document Header Definitions

Field

Description

Doc Nbr

The unique identifier for this Proposal Development document

Initiator

The user name of the person who created this Proposal Development document

Status

The state of the document as it relates to the completion of the document

KC sets this to In Progress when you create a Proposal Development document

Not related to submission status

Created

The time and date the document was created

 

For more information about the Document Header, see  E-Doc Fundamentals.
