The Intellectual Property Review document contains information about Intellectual Property (IP) reviews conducted at your institution.  IP reviews are required for proposals submitted to a commercial sponsor.  The Intellectual Property Review document facilitates the IP review process between the OSP and your institution’s Technology Transfer Office:

 

Figure 313 Intellectual Property Review Document

 

You create the Intellectual Property Review document when you are working in the Institutional Proposal document.  When you click on the Intellectual Property Review tab, KC takes you to the Intellectual Property Review document.  After completing and saving the Intellectual Property Review document, KC returns you to the Institutional Proposal.

 

For procedures for initiating, copying, saving, reloading, closing, canceling, and searching for Intellectual Property Review documents, see  E-Doc Fundamentals.

For procedural information on creating the Intellectual Property Review document from the Institutional Proposal document, see  Institutional Proposal Document, Intellectual Property Review Tab.

For procedural information about navigation from the Intellectual Property Review document back to the Institutional Proposal document, see  Intellectual Property Review Document, Return to Institutional Proposal.
