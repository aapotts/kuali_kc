Parameters Tab

The Parameters tab displays panels that show details of the budget version that is currently open.

Figure 214 Parameters Tab on a Budget Document

 

You must ensure that the correct budget version is on screen before you begin any data entry or modifications.  Check the document header for the Budget Name and Version #. 

Figure 215 Parameters Tab with the Budget Name and Version # Highlighted

 

If you do not have the correct budget version open, go to the Budget Versions tab, Budget Versions panel, and click  to access a different budget version.

Before you make any modifications to any budget, look at the document header and Version # to confirm you are working in the correct budget version.
