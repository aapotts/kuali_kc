Procedure for Adding an Employee Traveler’s Name Using the KcPerson Lookup

When you click  to find and select an Employee, KC displays the KcPerson Lookup screen:

 

Figure 402 KcPerson Lookup Screen

 

Enter appropriate information in the search fields to find the employee you need (or leave all the fields blank to display all employees), and then click  to display a list of Employees.  When you click search, KC displays the KcPerson Lookup screen again, with the results of your search displayed below the buttons:

 

Figure 403 KcPerson Lookup Screen with Search Information

 

KC displays the Employees that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, click return value on the row for the employee you need.  This enters that Employee in the Approved Foreign Travel sub-panel in the Special Approved panel.

When you click return value, KC displays the KC Award screen again with the selected employee’s name added to the Approved Foreign Travel sub-panel in the Traveler Name field:

 

Figure 404 Filled Approved Foreign Travel Sub-Panel
