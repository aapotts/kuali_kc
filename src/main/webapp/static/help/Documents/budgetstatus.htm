Budget Status

Use the Budget Status maintenance document to lookup, create, and update Budget Statuses.  This status is used to show if the Budget Version is Complete or Incomplete.  Budget Status appears on the Proposal Development e-doc, Budget Versions Tab, and Budget Development e-doc, Parameters Tab.

To access the Budget Status Lookup screen:

Maintenance main menu tab > Proposals menu group > Budget Status link
