Basic Search

KC displays the basic search on the Document Lookup screen when you click .  You can enter information into one or more of the search fields or leave all of the search fields blank to get a list of all of the documents in KC:

Date Created From

Optional

 

The first date on which the document you need may have been created

You use the Date Created From date with the Date Created To date to search for a document created within a specific date range.

 

You can leave the Date Created From field blank and only complete the Date Created To field if you want to find all documents created before the Date Created To date.

 

Enter the Date Created From in MM/DD/YYYY format or select the Date Created From by clicking .

Date Created To

Optional

 

The last date on which the document you need may have been created

You use the Date Created From date with the Date Created To date to search for a document created between those two dates.

 

You can leave the Date Created To field blank and only complete the Date Created From field if you want to find all documents created after and including the Date Created From date.

 

Enter the Date Created To in MM/DD/YYYY format or select the Date Created To by clicking .

Name this search (optional)

Optional

 

The name you want to give to this set of search criteria

You can name the search if you know you expect to use this set of search criteria frequently.

Enter a name for this search into the box.  You can use any combination of letters, numbers, special characters, and spaces.

After naming and running a search, the search name appears in the Searches .  You can retrieve these search criteria at any time by clicking the Searches .
