Budget Category Type

User the  Budget Category Type maintenance document to lookup, create, and update Budget Category Types.  This maintenance document allows  you to establish and edit types of budget categories and specify the sequential order in which they are sorted when made available for reference/selection in proposal & budget development. Budget Category Types appear as panels on the Budget e-doc, Non-Personnel tab.

To access the Budget Categories Lookup screen:

Maintenance main menu tab > Shared menu group > Budget Category Type link
