Edit Cost Element Tab

The Edit Cost Element tab consists of two required and two optional fields you use to define codes for objects that are selected as costs in research budgets.

Figure 572  Edit Cost Element

Table 263  Edit Abstract Types Tab components

Field

Description

Object Code Name

Required.  Enter the numeric value name of the object code.  This number uniquely identifies the object code.

Budget Category Code

Optional. Enter the code, or perform a lookup to return the code value. This will place the object code into a budget category (personnel, for instance).

Description

Required.  Enter the name of the object code.

On/Off Campus Flag

Required.  Check the box to have the object code default to On Campus when it is used in the budget development e-doc.

 

 

Edit existing values only if the new description is synonymous with the original term.  You can add new values. You should not delete existing values. Consider the impact on S2S submission and form mapping before making changes.
