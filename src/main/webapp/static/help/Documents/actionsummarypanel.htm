Action Summary Panel

The Action Summary panel provides an executive level summary of award funding obligations.

 

Click  on the Action Summary panel to displays its information:

 

Figure 371 Action Summary Panel

 

The Action Summary panel in the Time& Money tab contains read-only fields with data provided from other panels or sources.  Some fields in the panel may be blank.

 

Field

Description

Notice Date

Read only

 

The date of the underlying award notice

 

KC displays the Project Start Date from the Award Hierarchy panel as the Notice Date

Transaction Type Code

Read only

 

The reason for adding or modifying award data

 

KC displays the Project End Date from the Award Hierarchy panel

Obligation Start

Read only

 

The effective date of the current funding period

Obligation End Date

 

Read only

 

The end date of the current funding period

Obligated Cumulative

Read only

 

The total amount authorized by the sponsor for spending on the award to date, not including recipient cost sharing

Obligated Change

Read only

 

The increase or decrease in the Obligated Cumulative amount due to this transaction.
