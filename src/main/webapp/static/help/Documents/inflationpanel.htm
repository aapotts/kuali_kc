Inflation Panel

The Inflation panel shows which inflation rates apply to each Object Code Name in a budget version.  The Inflation panel may contain several dozen line items:

Figure 229 Inflation Panel on the Rates Tab

 

Open the Inflation panel on the Rates tab to see itemized default inflation rates and to check the Institute Rates. 

Manually entered Applicable Rates override the Institute Rates.

To override the default inflation rates for any line item on the Inflation panel, enter the override rate in the related Applicable Rate column.
