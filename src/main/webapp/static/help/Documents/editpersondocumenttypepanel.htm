Edit Person Document Type Panel

The Edit Person Document Type panel requires population of two fields that define an entry in the reference table, one a numeric value, and the other a textual description of the associated value.

Figure 584  Edit Person Document Type

 

Table 275  Edit Person Document Type components

Field

Definition

Person Document Type Code

Required.  Enter the value that will be used to represent this single piece of information when referenced.

Description

Required.  Enter the familiar name of the person document type you would use to refer to it verbally or in writing.

 

 

You may modify any of these existing values.  Evaluate the impact on S2S submissions before making changes.
