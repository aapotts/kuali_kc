Adding Personnel Details

The Add Details panel displays details related to the project person selected for the selected period.  A combination of Required and Optional fields sends information to KC to use in calculating budget information including, but not limited to Unrecovered F&A, Cost Sharing, and Inflation. 

The Personnel Detail panel has one section: Add Details.

The Add Details section has as many sub-panels as there are different Object Code Names for this budget version.  The Person drop-down list is at the top of each Object Code Name section (see Figure 245).
