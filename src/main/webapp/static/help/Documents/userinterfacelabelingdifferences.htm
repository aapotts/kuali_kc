User Interface Labeling Differences:

•         Proposal Development Document, Proposal Document, Proposal e-doc and Proposal refer to the same document in KC.  While every attempt has been made to use Proposal Development Document uniformly throughout this user guide, you may find they are used interchangeably within the KC user interface and the various types of KC documentation.

•         Create Proposal link on the Researcher menu tab and the Proposal Development  add button on the Central Admin menu tab accomplish the same thing:  both will initiate a new, blank Proposal document with a unique document number.

•        

•         Action List or Doc Search functions access saved (as opposed to new or unsaved) proposal documents.

For more information about accessing existing documents from action list or search result tables, see “Kuali Enterprise Workflow”.
