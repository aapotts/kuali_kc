Account Type Lookup

When you click the Account Type link in the Awards menu group on the Maintenance main menu tab, KC displays the Account Type Lookup screen:

 

Figure 444 Account Type Lookup Screen

 

Use the Account Type Lookup screen to search for and display existing Account Type Codes.

Enter appropriate information in the search fields to find the Account Type Code you need (or leave all the fields blank to display all Account Type Codes), then click  to display a list of Account Type Codes.  When you click search, KC displays the Account Type Lookup screen again, with the results of your search displayed below the buttons:

 

Figure 445 Account Type Lookup Screen with Search Information

 

KC displays the Account Type Codes that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, you can either edit or copy an Account Type Code or Description using the links in the Actions column.

For information about editing an existing maintenance document such as an Account Type, see Editing A Maintenance Document in Maintenance Document Operations.

For information about copying a maintenance document such as an Account Type, see Copying A Maintenance Document in Maintenance Document Operations.

When your institution creates a new Account Type Code, it enters a Description for that code.

The Account Type Lookup screen has a  button in the top right corner that lets you create a new Account Type.

See the Account Type Screen: Panels section below for additional information about creating a new Account Type.
