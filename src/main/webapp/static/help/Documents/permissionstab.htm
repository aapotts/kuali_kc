Permissions Tab

The Permissions tab contains the users who have access to a proposal for purposes of creating that proposal.  You can add or remove users from the proposal creation process on this tab.  You assign each user to a role.  That role determines whether the user can view or modify the proposal details, the proposal narrative, and the budget:

 

KC automatically assigns the users associated with the proposal’s Lead Unit on the Permissions tab.  You specify the Lead Unit on the Required Fields for Saving the Document panel on the Proposal tab.

The user and role information on the Permissions tab populates user fields on the Abstracts and Attachments tab and in the Budget module.
