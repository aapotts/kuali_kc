Notes Panel

TheNotes panel is a common e-doc panel.  You use it to enter internal notes about the document:

 

Figure 114 Notes Panel

 

The Notes panel replaces the Notes and Attachments panel in some e-docs when the e-doc contains other panels for adding attachments.  For example, the Proposal Development e-doc uses the Notes panel because this e-doc contains other panels for adding attachments.

For more information about the Notes and Attachments panel, see the  Common E-Doc Tabs and Panels, Notes and Attachments Panel.

 

Table 29 Notes Panel Definition
