Procedure for Adding a Proposal to an Award:

When you click  to find and select a Proposal Number, KC displays the Proposal Lookup screen:

 

Figure 326 Proposal Lookup Screen

 

Enter appropriate information in the search fields to find the Proposal Number you need, or leave all the fields blank to display all Proposal Numbers, and then click  to display a list of Proposal Numbers.  When you click , KC displays the results of your search below the search button:

 

Figure 327 Proposal Lookup Screen with Search Information

 

KC displays the Proposal Numbers that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, click return value for the Proposal you need to add to this Award.

KC displays the KC Award screen again with the selected Proposal Number in the Add Funding Proposals section:

 

Figure 328 KC Award, Funding Proposal Panel with Proposal Number Entered

 

When you click in the Add Funding Proposals section, KC displays this Proposal in the Current Funding Proposals section of the Funding Proposals panel.  KC also makes available the Details sub-panel for this Proposal.
