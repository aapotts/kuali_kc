Notes & Attachments

The Notes & Attachments page of the Protocol document allows for the upload of protocol and personnel attachments, as well as the ability to enter textual notes.  You are able to describe the attachment and specify the type of attachment.  Attachments can be of various recognized types (for example, Informed Consent or Recruitment Brochure) or miscellaneous.  Any type of document or file format is supported.  Any user with viewing access to the Protocol document can view attachments and notes.  Version history for multiple attachments with the same filename is preserved. 

In a future release, the ability to attach watermarks for PDF format printing/viewing will also be possible from this page.
