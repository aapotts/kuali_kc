Project Income Panel

The Project Income panel is where you detail anticipated project income by project period and source.

Figure 258  Project Income Panel

 

Project income is the gross university earnings generated by a sponsored activity or earned as a result of an award.  KC includes a mechanism for collecting anticipated project income.  For some grants.gov submissions, the project income amount will be reported at the time of proposal submission.

The sponsor may elect to reduce the award based on anticipated project income or allow the project income to be used as additional support for the project.  Specific instructions regarding the disposition of project income should appear in the sponsor’s program announcement and/or award guidelines.
