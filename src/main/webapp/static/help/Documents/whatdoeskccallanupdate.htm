What does KC call an update?

•      When one or more field entries change anywhere in the Budget Document, KC calls this an update.

•      If you click in any text field, radio button, or checkbox on any panel of any Budget Document tab, KC calls this an update – even if you change nothing.

Example: If you click in a Calendar field but do not change the date, KC calls this an update because you clicked in the Calendar field.
