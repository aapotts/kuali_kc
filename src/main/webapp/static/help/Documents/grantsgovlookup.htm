Grants.gov Lookup

The Grants.gov lookup in KC searches the www.grants.gov website for opportunities.  The search uses your proposal’s CFDA Number and/or Opportunity ID. 

 

1.  To search for opportunities, click :

 

     Figure 131 Grants.gov Lookup

 

          KC then displays the Opportunity Lookup screen:

 

          

                 Figure 132 Opportunity Lookup Screen

 

The Opportunity ID and/or CFDA Number on the Opportunity Lookup screen default to the information you entered on the Sponsor & Program Information panel.  You can change the Opportunity ID or CFDA Number on the Opportunity Lookup screen in step 2.  If you do not want to change the Opportunity ID or CFDA Number, proceed to step 3.

 

2.  Enter an Opportunity ID and/or CFDA Number.

 

3.  Click .

 

KC displays the Opportunity Lookup search results list:

 

     Figure 133 Opportunity Lookup Search Results List

 

If you click  without entering an Opportunity ID or a CFDA Number, KC displays this error:

 

Enter an Opportunity ID or a CFDA Number and click  again.

 

4.  Click the return value link to select an opportunity for which you want to apply:

 

        

                   Figure 134 Return Value Link

 

If you select an opportunity that is not a valid S2S candidate, KC displays this error: 

Enter an Opportunity ID or a CFDA Number and click  again.

The remove opportunity <need icon> button appears after you select an opportunity.

KC fills in the Opportunity sub-panel with the information for the opportunity you select.

KC also populates the Forms Sub-panel after you select an opportunity.

KC populates the Opportunity ID, the Opportunity Title, and the CFDA Number on the Sponsor & Program Information panel after you select an opportunity.  KC replaces the information you previously entered into these fields on the Sponsor & Program Information panel.

If you change the Opportunity ID, the Opportunity Title, or the CFDA Number on the Sponsor & Program Information panel after KC populates these values, KC displays an error during data validation.
