Proposal Page Tab

 The Proposal page tab of the Proposal Development Document is where basic proposal information is entered.  It displays the first set of document panels for user input when a new proposal is begun.  The first two panels, Document Overview and Required fields for Saving Document, indicate the data fields that must be populated in order to save a new proposal that may subsequently be retrieved for completion and submission.
