Tabs

The largest divisions of a KC document is a tab. The tabs of a document should not be confused with the Main Menu tabs. Throughout the User Guide volumes 1 and 2, document tabs are referred to as tabs. You might think of a filing system in which a page in the file is marked by a raised tab. However, for simplicity, this documentation does not distinguish between a tab and the page it represents. All pages of a document are referred to as tabs.
