Description

Budget Overview

This panel displays the project start/end dates and allows you to:

•      Change the budget version status

•      Specify whether the budget version is the final budget

•      Specify whether it is a modular budget

You can also enter comments, a total cost limit, residual fund amounts, and identify rate types on this panel.

Budget Periods and Totals

This panel allows you to sub-divide the budget into budget periods by specifying start and end dates for each period.

You may add costs directly to the panel in the form of a summary budget. 

KC creates budget periods based on the project start and end dates that you enter on the proposal.  It assumes that each budget period is one year in length unless you change the budget period(s).
