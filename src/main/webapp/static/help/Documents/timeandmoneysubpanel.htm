Time and Money Sub-Panel

The Time & Money sub-panel includes the Project Start Date, Project End Date, Execution Date, Anticipated Amount, Obligation Start Date, Obligation End Date, and Obligated Amount fields.

Users with Administrator authorization may enter information in these fields.

 

Figure 342 KC Award, Dates & Details Panel, Time& Money Sub-Panel

 

Table 126 KC Award, Dates & Details Panel, Time & Money Sub-Panel Field Descriptions

Field

Description

Project Start Date

Optional

 

The project period start date based on the sponsor award instrument

 

This may not be the same as the Project Start Date in the Institutional Proposal document.

 

Enter the Project Start Date in MM/DD/YYYY format or select the Project Start Date clicking .

Project End Date

Required

 

The expiration date of the current funding period, plus unfunded periods (it may include possible option years)

 

Enter the Project End Date in MM/DD/YYYY format or select the Project End Date clicking .

Execution Date

Optional

 

The date the sponsor award instrument is fully signed

 

Enter the Execution Date in the MM/DD/YYYY format or select the Execution Date clicking .

Anticipated Amount

Optional

 

The anticipated sponsor funding for the entire project period: This is the current obligation plus anticipated incremental funding and future year funding, including possible option years.  It does not include cost share.

 

Editable entry field for simple award hierarchies (one node); display-only field for complex award hierarchies (more than 1 node)

 

Enter the Anticipated Amount into the box.

Obligation Start Date

Optional

 

The effective date of the current funding period

 

This must be the same as or later than the Project Start Date.

 

Enter the Obligation Start Date in MM/DD/YYYY format or select the Obligation Start Date clicking .

Obligation End Date

Optional

 

The end date of the current funding period

 

This must be the same as or earlier than the Project End Date.

 

Enter the Obligation End Date in MM/DD/YYYY format or select the Obligation End Date clicking .

Obligated Amount

Read only

 

The amount obligated by the sponsor to date under the award.  It does not include cost share.

 

Obligated Amount must be equal to or less than the Anticipated Amount.

Editable entry field for simple award hierarchies (1 node); display-only field for complex award hierarchies (> 1 node).

 

Enter the Obligated Amount into the box.
