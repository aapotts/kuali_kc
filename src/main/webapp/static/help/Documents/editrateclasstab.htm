Edit Rate Class Tab

The Edit Rate Class tab is comprised of three required fields that collectively make up a single reference table record.

Figure 634  Edit Rate Class

 

Table 291  Edit Rate Class components

Field

Definition

Rate Class Code

Required.  The numeric value that the system uses to reference this rate class.

Rate Class Type

Required.  Enter the alphanumeric type of rate class this code is a part of, or select it from the lookup  or direct inquiry features.

Description

Required.  The name you use to refer to the rate class verbally or in writing that also uniquely identifies it from others.  Enter text as appropriate (any character or spacing is acceptable).
