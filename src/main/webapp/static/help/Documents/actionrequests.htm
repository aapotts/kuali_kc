Action Requests

KC sends action requests to approvers when you submit a document. 

If you receive an action request, the action request type will display in your Action List.

If you route a document to an ad hoc reviewer, you will select the type of action you are requesting.

 

There are three types of KC action requests: 

 

Table 152 Action Requests

Action Requests

Description

Approval

You must review the document and approve or disapprove its contents to complete the action request.

You must complete the Approval action request in order for the workflow to proceed

Acknowledge

You must review the document and acknowledge its contents.

Workflow routing will proceed with uncompleted Acknowledge action requests.

FYI

You must indicate receipt of the document.

Workflow routing will proceed with uncompleted the FYI action request.
