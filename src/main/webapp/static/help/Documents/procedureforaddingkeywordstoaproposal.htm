Procedure for Adding Keywords to a Proposal

Adding a keyword to a proposal associates that keyword with the proposal for future look up and reporting on a specific proposal topic.  You can add keywords to a proposal using (select)  in the Add: row at the top of the Keywords panel:

 

 

Figure 108 Keywords Select

 

Clicking (select)  takes you to the Keyword Lookup screen. You enter the Keyword Code or keyword Description on the Keyword Lookup screen to search for keywords from the keyword list. 

•      The Keyword Code identifies the keyword using a combination of letters and numbers assigned by KC or your institution. 

•      The Description is the keyword or keywords. 

You may leave both the Keyword Code and Description fields blank to display your institution’s entire keyword list. 

 

Once you’ve found the correct keywords, you select and return them so that KC associates them with this proposal.

 

1.  Click  in the Add: row of the Keywords panel to display the Keyword Lookup screen:

         

     Figure 109 Lookup Icon on the Add Row

 

         KC displays the Keyword Lookup screen:

          

      

     Figure 110 Keyword Lookup Screen

 

2.  Enter a Keyword Code and/or a Description for the keyword(s) that you need to add to this proposal and click  to display those keywords.

 

KC displays the Keyword Lookup search results list:

     

     Figure 111 Keyword Search Results List

 

3.  Click  next to each keyword that you need to associate with this proposal.  Click  to check mark all of the keywords in this list.  Click  to clear all of the checkboxes in this list.

 

4.  Click  to add these keywords to this proposal and return to the Keywords panel.

5.  KC displays the keywords you selected in the Keywords panel for this proposal:

        

     Figure 112 Keywords Panel
