Object Code

Use the Object Code maintenance document to lookup, create and update  cost elements that are used on proposal development screens for reference and selection in preparation of research budgets.  They allow for specification of expenses by category. The terms object codes and cost elements are used interchangably.  You select object codes while preparing a budget in the KC Budget Development e-doc.
