Edit Template Contact Panel

The Edit Template Contact panel lets you create one or more New Sponsor Template Contacts.  These contacts are for common award maintenance tasks like financial and technical report submission.  When synchronized with an award, the Sponsor Template updates the award with the contact information.

The Edit Template Contact panel lets you select a Contact Type Code and Rolodex Id for each New Sponsor Template Contact.
