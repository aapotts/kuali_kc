Edit Science Keywords Panel

The Edit Science Keywords tab allows you to specify a single keyword reference table entry at a time by defining the numeric value and associated textual word.

 

Figure 547  Edit Science Keywords

 

Table 244  Edit Science Keywords components

Field

Definition

Keyword Code

Required.  A numeric value associated with the term that describes the discipline of a sponsored project (e.g.  sustainable agriculture, children's literature, etc.). 

Description

Required.  Enter the text that expresses what the science keyword code refers to.  A term that describes the discipline of a sponsored project (e.g.  sustainable agriculture, children's literature, etc.). 

 

 

You may modify any of these existing values.
