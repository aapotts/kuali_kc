Person Document Type

Use the Person Document Type maintenance document to lookup, create, and update Person Document Types.  These document types are related to persons added to a proposal, and include common formats such as Biographical Sketches and Current & Pending Support pages. You will see Person Document Types on the Personnel Attachments panel of the Abstracts and Attachments tab in KC Proposal Development.

 

To access the Person document Type Lookup screen:

Maintenance main menu tab > Shared menu group > Person Document Type link
