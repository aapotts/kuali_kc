Submission History Sub-Panel

The Submission History sub-panel contains details about the submission of your proposal to www.grants.gov.  KC fills in the Submission History sub-panel after you submit the proposal to www.grants.gov.

 

Figure 140 Submission History Sub-Panel

 

Table 42 Submission History Sub-Panel Definition

Column

Description

 

Posted Timestamp

Read only

The date and time you submitted this proposal

 

Submitted By

Read only

The KcPerson Id of the person who submitted the proposal to www.grants.gov

 

Submission Type

 

Read only

The Submission Type selected on the Opportunity sub-panel

 

Revision Type

 

Read only

The Revision Type selected on the Opportunity sub-panel

 

Federal Identifier

Read only

The tracking number assigned by www.grants.gov when you successfully submitted this proposal

 

Original Proposal Id

 

Read only

The www.grants.gov identifier for the original approved proposal

 

KC only displays an Original Proposal ID for applications with a Submission Type of Change/Corrected Application.

 

For more information about how to use the action buttons at the bottom of this tab, see Kuali Coeus 2.0 User Guide, Kuali Enterprise Workflow, Workflow Action Buttons.
