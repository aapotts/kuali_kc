Rate Classes sub-panel (on the Object Code Name sub-panel)

Click  to expand the sub-panel.

The Rate Classes sub-panel displays selected Rate Costs and Rate Cost Sharing totals for the Object Code Name. 

Figure 248 Rate Classes Sub-Panel on the Object Code Name Sub-Panel

 

Table 79 Rate Classes Sub–panel

Field

Description

Rate Class

Broad category of rates that are applied to expenses in the KC budget 

F&A Rates, Fringe Benefits, Inflation, etc. are rate classes. 

A single rate class may have multiple rate types. 

Rate Type

 

Rate types are more detailed than rate classes.  They form subsets of rate classes.

When no sub–division is required, the Rate Type name may be similar to the Rate Class name.

Apply Rate?

For each line item displayed on the Rate Classes panel, select which rates are active when KC performs calculations.

 denotes Yes to the question, Apply Rate?

Rate Cost

Total cost to the sponsor for all personnel in the Object Code Name/Group with the Rate Class and Rate Type indicated

Rate Cost Sharing

Total shared cost (to someone other than the sponsor) for all personnel in the Object Code Name/Group with the Rate Class and Rate Type indicated
