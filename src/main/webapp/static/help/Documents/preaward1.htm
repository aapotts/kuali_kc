Pre-Award

The Pre-Award group on the Unit menu provides convenient, efficient access to KC functions that are commonly used by Unit users.  It contains “create new” and “lookup existing” icon buttons that take you to new, blank e-docs or lookup screens that allow you to search for e-docs, respectively.

Figure 83 Unit Menu – Pre-Award Group

The Pre-Award group contains four functions, each with “create new” and “lookup existing” icon buttons.  Each is summarized in the table that follows, along with crossreferences to more detailed instructions on how to use the screens they take you to.

Table 25 Unit Menu, Pre-Award Group - Function Descriptions

Description (& link to detailed instructions)
