F & A Panel Header

The F & A panel displays the proposal Activity Type as the first word in the panel heading.  In the screen print above, the Activity Type is Instruction.  You can set the proposal Activity Type on the Required Fields for Saving Document panel on the Proposal tab of the Proposal Development Document:

 

Figure 223 The Activity Type Drop-down List on the Proposal Tab of the Proposal Development Document

 

The person who first creates a proposal selects an Activity Type, as shown in the example above.  That Activity Type appears in the header of the F & A panel.
