Single Value Lookup

This function (sometimes referred to as search or inquiry and maintenance) lets you use search criteria to locate values for populating an alphanumeric field.  You may search using full, partial, or no knowledge of the item sought.  The Lookup feature helps you save time and reduce data entry errors.

Access Lookup screens in two ways:

1.  click the Lookup icon

2.  many of the links on the Main Menu tabs will open a Lookup
