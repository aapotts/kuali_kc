Modular Budget Overview

The Modular Budget Overview (Project) section of the Modular Budget page displays three fields of key information about the modular budget – Project Start Date, Project End Date, and Project Total Requested Cost.  The information displayed is based on your selection in the Select Modular Budget Period bubble area.

Figure 248 Budget Document > Modular Budget Page > Modular Budget Overview (Project) Section – Example Layout

 

Table 139 Budget Document > Modular Budget Page > Modular Budget Overview (Project) Section - Field Descriptions

Field

Description

Project Start Date

The start date of the specific budget period being viewed.

Project End Date

The end date for the displayed period.

Project Total Requested Cost

Sum of all period total requested costs from each budget period view.  Total requested amount for the entire project period, including direct costs, F&A costs, and Consortium F&A Costs.
