Budget Versions Panel

In KC, each budget attached to a Proposal Development Document is a Budget Version.  You may create and add multiple budget versions to a Budget Document.

 

Figure 213 Budget Versions Panel

 

Each budget version has its own sub-panel on the Budget Versions panel.  In the example above, you can see two budget versions, named budget 1 and budget 2.

 

Table 67  Budget Versions Panel Field Descriptions and Procedures

Field or Button

Description

Panel Label

The start and end dates of the proposal appear in parentheses.

Name

Required

 

The user–supplied name of the budget version: This does not have to be unique.  (KC assigns a unique number to each budget version when you first add it to the Budget Document.)

The budget name can be up to 40 characters long.

If you enter more than 40 characters in this field, KC displays an error message when you try to add this budget version.

Version #

Sequential system-assigned number that identifies each budget

Direct Cost

 

Shows the total Direct Costs entered on the Budget Periods and Totals panel on the Parameters tab

F&A

 

Shows the total Facilities and Administration (F&A) costs entered on the Budget Periods and Totals panel on the Parameters tab

Total

Shows the sum of the Direct Costs and F&A costs entered on the Budget Periods and Totals panel on the Parameters tab

Budget Status

Read-only

Shows the budget version status as selected on the Budget Overview panel on the Parameters tab

Only one budget version in a proposal may be marked Complete.

Final

Checking this box indicates that the budget version is the Final version for the proposal.

A proposal may have only one Final version.
