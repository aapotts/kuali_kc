Deleting Personnel from a Proposal

When you delete an individual on the Key Personnel tab of the Proposal Development Document, that name stays on the Project Personnel panel, but KC does make changes.  KC removes that person’s Job Code and the key personnel label in parentheses next to their name. 

You now have two choices:

1.  Click  to remove that person from the Project Personnel panel.

2.  Select a new Job Code for the person.
