Comments, Notes & Attachments Tab

The Comments, Notes & Attachments tab provides a shared location for your institution to write and store information about the project for future reference.

Click  on the panel to display its information:

 

Figure 94 Comments, Notes & Attachments Tab Panels

 

The Comments, Notes & Attachments tab includes Comments, Notes, and Attachments panels for retaining reference information.

 

Table 24 Comments, Notes & Attachments Tab Panel Descriptions

Panel

Description

Comments

Add comments about specific topics on the project or award and sync comments to an existing sponsor template that contains default comment values

Notes

Write and save references about the project or award

Restricted notes can only be created and seen by OSP staff

Attachments

Attach files for reference about the project or award
