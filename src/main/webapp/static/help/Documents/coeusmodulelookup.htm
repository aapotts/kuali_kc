Coeus Module Lookup

When you click the Module Names link in the Miscellaneous menu group on the Maintenance main menu tab, KC displays the Coeus Module Lookup screen:

 

Figure 554 Coeus Module Lookup Screen

 

Use the Coeus Module Lookup screen to search for and display existing Module Codes.

Enter appropriate information in the search fields to find the Module Code you need (or leave all the fields blank to display all Module Codes), then click  to display a list of Module Codes.  When you click search, KC displays the Coeus Module Lookup screen again, with the results of your search displayed below the buttons:

 

Figure 555 Coeus Module Lookup Screen with Search Information

 

KC displays the Module Codes that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, you can either edit or copy the Module Code or Description using the links in the Actions column.

For information about editing an existing maintenance document such as a Module Names, see Editing A Maintenance Document in Maintenance Document Operations.

For information about copying a maintenance document such as a Module Names, see Copying A Maintenance Document in Maintenance Document Operations.

When your institution creates a new Module Code, it enters a Description for that code.

The Coeus Module Lookup screen has a  button in the top right corner that lets you create a new Module Code.

See the Kuali Coeus Module Screen: Panels section below for additional information about creating a new Module Code.
