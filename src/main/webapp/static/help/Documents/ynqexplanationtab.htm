Ynq Explanation Tab

 

The Ynq Explanation tab is what you use to explain the rationale or necessity of requiring the question be answered.

 

Figure 732  Ynq Explanation

 

Table 336  Ynq Explanation components

Field

Description

Explanation

A classification of question related to explanation of the background information necessary to describe the purpose of the question being asked.

Policy

The institutional policy statement(s) that require the question be asked.

Regulation

The description of the applicable regulation as it relates to the question.
