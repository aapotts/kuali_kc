Inflation Default Rates

You must set the inflation rate every fiscal year.  KC will not use the last known rate to set inflation rates for a proposal.  If KC does not find an inflation rate set for the current fiscal year, it defaults to 0% until you set a rate for the current fiscal year. 

 

You must set default rates using an Institute Rates Maintenance Document. 

 

You can set override rates using only the panels on the Rates tab.
