Filter

You use the Filter to condense the number of action requests KC includes in your Action List.  For example, you may want to display only those action requests for Proposal Development documents.  You can set up a Filter to display only Proposal Development document types.

You set up the Filter by entering information you want KC to match for the Filter.  When you save the Filter fields, KC checks the information you saved against your action requests.  KC includes the matched action requests in your Action List. 

You can exclude the matched action requests from your Action List.  For example, you may want exclude action requests with a Route Status of Complete.  When you designate an exclusion, KC displays all of your Action List items, except for those that match the Filter fields.
