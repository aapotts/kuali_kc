Keywords Panel

A keyword describes the discipline or subject matter of a sponsored project (i.e. sustainable agriculture or children's literature).  Keywords may be single words or phrases. 

Keywords are used as search parameters to locate a specific proposal in university databases.  Once located, the proposal may provide needed data; it may be edited or copied.

In proposal development, Keywords are selected from a Keywords Table.  These values can be institution-specific or lists of general terms from which multiple projects might select keywords.

Keyword Code:

Enter the code of the keyword if you have it.  When you select search, the word associated with that code will be identified and made available for selection.

If the Keyword Code field is empty and the Description field is also empty, selecting search will retrieve all keywords available.

Description

This field works as a simple search engine in which an asterisk * serves as a search wildcard.  Example: Entering ostra* might return Ostracods for selection as a keyword.  You must enter at least one alphanumeric character followed by an asterisk in order to search.  As of KC 2.0, this search mechanism recognizes only one entry at a time.  Boolean operators are not recognized.

Actions

      Initiates your search using the criteria entered in the Keyword Code and Description fields.  No criteria entered will retrieve all Keywords available.

          Clears all data entered in the Keyword Code and Description fields

      Cancels the search and returns to the Keywords tab.

Select?

To select a keyword for use, left click in the checkbox in the row aligned with that keyword.

Actions

   Selects all keywords displayed by placing a checkmark in all checkboxes.

           Unselects all keywords by removing any checksmarks displayed.

      Identifies all keywords selected and places them in the keywords list on the Keywords tab.

Selecting  sorts keywords by the corresponding column content – Keyword Code or Description.
