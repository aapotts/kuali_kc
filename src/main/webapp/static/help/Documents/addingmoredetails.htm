Adding More Details

To add more detail for a specific individual you must first select the person from the Person drop-down list, then assign an Object Code Name, assign a Group if appropriate, then click  to complete setting that person up on the Personnel Detail panel (see the "Add Details Section" above.)

1.  Find the sub-panel with the Object Code Name assigned to the individual for whom you want to add details.

2.  Click  to open the sub-panel.

Each Object Code Name used in the budget has its own sub-panel by the same name on the Personnel Detail panel.  Each of these sub-panels displays a pair of its own sub-sub-panels: Details and Rate Classes.  The data entered on these two sub-sub-panels applies to all persons listed under the Object Code Name appearing on the sub-panel's header.

 

Subtopics:

Object Code Name sub-panel

Details sub-panel (on the Object Code Name sub-panel)

Rate Classes sub-panel (on the Object Code Name sub-panel)
