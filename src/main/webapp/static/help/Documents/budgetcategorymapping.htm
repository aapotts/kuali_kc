Budget Category Mapping

The Budget Category Mapping maintenance document allows you to map Coeus category codes with KC codes and save them as individual maps that can be reused throughout the system.  It enables you to map KC budget category codes to each of the sponsor-specific form categories defined with the Budget Category Maps maintenance document.

Prior to modifying or deleting existing values, it is recommended that you first assess the impact on electronic proposal submissions.

Edit Budget Category Mapping Section

The Edit Budget Category Mapping section of the Budget Category Mapping maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 417 Budget Category Mapping Maintenance Document - Edit Budget Category Mapping Section Field Descriptions

Field

Description

Budget Category Code

Required.  A numeric value that uniquely identifies the Budget Category.  Type the suspected value in the box and click the direct inquiry icon or click the lookup icon to search for and return the value to populate the box automatically.

Mapping Name

Required.  The name of the mapping used to map the Budget Category Code to the target category code.

Target Category Code

Required.  The unique category code in KC that the Budget Category Code is being mapped to.
