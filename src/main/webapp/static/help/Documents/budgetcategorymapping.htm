Budget Category Mapping

Use the Budget Category Mapping maintenance document to lookup, create, and update Budget Category Mapping. Budget category mapping allows you to map KC budget category codes to each of the sponsor-specific form categories defined with the Budget Category Maps maintenance document (documented in the next section).

 

To access the Budget Category Mapping Lookup screen:

Maintenance main menu tab > Shared menu group > Budget Category Mapping link
