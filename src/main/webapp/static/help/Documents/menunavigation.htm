Menu Navigation

KC 2.0 displays five Main Menu Tabs:

1.  Researcher

2.  Unit

3.  Central Admin

4.  Maintenance

5.  System Admin

Each of these tabs displays several groups of KC functions.  These groups of functions are called menu groups; they help to arrange functions in a logically.
