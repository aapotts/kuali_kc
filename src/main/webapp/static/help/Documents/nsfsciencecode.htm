NSF Science Code

Use the NSF Science Code maintenance document to lookup, create, and update NSF Science Codes.  The NSF Science Code is a numeric code that is used by the institution to indicate a field of research activity, as defined by the National Science Foundation.  For example, the NSF science code for Chemical Engineering is 012.  These codes are important when an institution compiles data for the NSF Annual Survey, and for other annual reporting exercises.  The NSF is a federal agency that sponsors fundamental research in science and engineering and also supports science education.  You select the NSF Science code on the KC Proposal Development e-doc, on the Sponsor & Program Information panel of the Proposal tab.

 

To access the NSF Science Code Lookup screen:

Maintenance main menu tab > Shared menu group > NSF Science Code link
