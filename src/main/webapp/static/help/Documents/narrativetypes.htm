Narrative Types

Use the Narrative Type maintenance document to lookup, create, and update Narrative Types. Narrative types refer to the types of files attached to a proposal document.  These files are transmitted to the sponsor during Grants.gov S2S submission. For instance, Project Summary is a Narrative Type.  You select Narrative Types while uploading file attachments on the Abstracts and Attachments tab of the KC Proposal Development e-doc.

 

To access the F&A Rate Types Lookup screen:

Maintenance main menu tab > Proposals menu group > Narrative Types link
