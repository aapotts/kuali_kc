Two Sub-panels

For each line item entered on the Cost Category panels, there is a Line Item Details sub-panel that, when expanded, reveals a Rate Classes sub-panel. 

The Line Item Details sub-panels look the same for each line item on every Cost Category panel.  The content of every Line Item Details sub–panel is unique.

Each Rate Classes sub-panel shows only rate classes that apply to the line item Object Code Name.

Cost Sharing

Line item costs not billed to the project

Example:  The project needs two laser microscopes for the first of three budget periods.  Each costs 30,000 dollars for a total cost of $60,000.  Your institution decides to charge 1/3 of the cost to the project and pay remaining costs from university accounts.  $20,000 is the Total Base Cost and the remaining $40,000 is Cost Sharing.

Rate types are more detailed than rate classes.  They form subsets of rate classes. 

When no sub–division is required, the Rate Type name may be similar to the Rate Class name.
