Edit Person-Role Tab

The Edit Person-Role tab requires you provide valid values for 5 fields that make up a reference table record used in proposal development.

 

Figure 586  Edit Person-Role

 

Table 277  Edit Person-Role components

Field

Description

Person Id

Required.  Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions to specify the value that uniquely identifies the person.

Role Id

Required.  Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions to specify the value that uniquely identifies the role.

Unit Number

Required.  Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions to specify the numeric value that uniquely represents the institutional unit.

Descend

Required.  Click within the checkbox  to select the item to make the person-role record descend.  A green check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

Active

Required.  Click within the checkbox  to select the item to make it active for use in the system.  A green check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.
