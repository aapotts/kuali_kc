Frequency Base Lookup

When you click the Frequency Base link in the Awards menu group on the Maintenance main menu tab, KC displays the Frequency Base Lookup screen:

 

Figure 520 Frequency Base Lookup Screen

 

Use the Frequency Base Lookup screen to search for and display existing Frequency Base Codes.

Enter appropriate information in the search fields to find the Frequency Base Code you need (or leave all the fields blank to display all Frequency Base Codes), then click  to display a list of Frequency Base Codes.  When you click search, KC displays the Frequency Base Lookup screen again, with the results of your search displayed below the buttons:

 

Figure 521 Frequency Base Lookup Screen with Search Information

 

KC displays the Frequency Base Codes that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, you can either edit or copy the Frequency Base Code or Description using the links in the Actions column.

For information about editing an existing maintenance document such as a Frequency Base Code, see Editing A Maintenance Document in Maintenance Document Operations.

For information about copying a maintenance document such as a Frequency Base Code, see Copying A Maintenance Document in Maintenance Document Operations.

When your institution creates a new Frequency Base Code, it enters a Description for that code.

The Frequency Base Lookup screen has a  button in the top right corner that lets you create a new Frequency Base Code.

See the Frequency Base Screen: Panels section below for additional information about creating a new Frequency Base Code.
