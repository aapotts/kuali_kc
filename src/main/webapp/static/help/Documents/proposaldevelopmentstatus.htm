Proposal Development Status

Menu group link text:  Proposal Development Status

Lookup screen label:  Proposal State Lookup

 

The Proposal Development Status maintenance document link takes you to the Proposal State Lookup screen, which does not display a create new button.  This functionality allows you to view the possible states a Proposal Development document can be in, and see the corresponding numeric value associated with each.
