Screen Layout & Navigation

Most KC screens have design constants such as a header, body, and footer.  The header displays the KC logo on the left, menu tabs in the center, and a feedback e-mail link on the top, right.  A horizontal toolbar separates the header from the body with workflow action buttons on the left, the logged in user name to the right of center, and a login tool that allows for impersonation of other user roles (used for system diagnosis and training demonstration).  The footer displays copyright information and the small "k" Kuali logo.  While the header and footer remain static, the body is dynamic.

Each Main Menu tab offers a unique display in the body of the screen.  Each active link on a Main Menu tab leads to a specific location in the system.  The  Create Proposal link leads to a new Proposal Development Document. The Create Award link leads to a new Award Document.  From the Maintenance Main Menu tab, most links lead to a Lookup for the item named in the link title and a create new button is displayed with every Lookup accessed from the Maintenance Main Menu tab.
