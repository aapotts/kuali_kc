Procedure for Creating and Printing the Current and/or Pending Support Report

You print the Current or Pending Support reports by initiating the report and then printing it.  You may also export the report in CSV, XLS, or XML format.

1.  On the Current or Pending line, enter the person’s KcPerson Id.  If you do not know the KcPerson Id, click  to find and select one.

 

KC displays the person’s name on the Print Reports panel:

 

Figure 200 Person's Name

 

2.  Click  to create the report.

 

KC displays the report:

 

Figure 201 Pending Support Report

 

For more information about the fields listed on the Current and Pending Support reports, see  Institutional Proposal.

 

3.  Click  to print the report in PDF format or to save or send it to the sponsor.

4.  Click CSV, spreadsheet, or XML to export the report so you can edit, save, or send it to the sponsor.

For more information about exporting the report, see  E-Doc Fundamentals.
