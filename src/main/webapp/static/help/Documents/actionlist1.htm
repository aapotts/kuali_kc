Action List

The Action List displays all documents that have an outstanding action request for you to complete.  KC displays the columns and rows according to your preferences and filter.  In general, the Action List displays the document identification, the routing status and the action requested of you:

Action Requested

The action you are requested to perform on this document

The KC action requests are:

Approve:  You must review the document, and approve or disapprove it.

Acknowledge:  You must open the document and acknowledge it.  Pending acknowledgement actions interrupt workflow.

FYI:  You acknowledge the submission of the document.  You are not required to open the document.  Pending FYI actions do not interrupt workflow.

Complete:  Your action request is complete.

 

For more information about Action Requested, see  Kuali Enterprise Workflow, Ad Hoc Routing.
