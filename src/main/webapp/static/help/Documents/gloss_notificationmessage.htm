Notification Message

The overall Notification item or Notification Message that a user sees when she views the details of a notification in her action list. A Notification Message contains not only common elements such as Sender, Channel, and Title, but also content-type-specific fields.  This term represents the overall notification item or notification message that a user would see when she views the details of a notification in her action list.
