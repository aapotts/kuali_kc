Browser Navigation Warning

Whether attempting to navigate from screen to screen, menu to menu, or tab to tab, using your Web browser’s navigation tools (Back, Forward, and Refresh buttons typically found in the upper-left corner of the screen) can cause serious errors in your save and login / logout processes, the most serious being lost work and corrupted documents.  Your web browser’s standard navigation tools are not compatible with KC for navigation or any other function while you work within the KC environment.

In the event you use the browser navigation tools (i.e.  back arrow, etc.), you may not notice any adverse effect.  However, it is recommended that you log out completely, close your web browser, and open a new browser window, and log in again.  DO NOT attempt to undo hidden or visible damage by using the refresh command or another browser navigation tool.
