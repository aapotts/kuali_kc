Edit Sponsor Form Templates Tab

Edit the reference table record by specifying the four required fields on the Edit SponsorFormTemplates tab and using the Browse feature to upload and include the form itself in the form of an attachment.

 

Figure 668  Edit SponsorFormTemplates

 

Table 312  Edit SponsorFormTemplates components

Field

Definition

Package Number

Required.  Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions.

Page Description

Required.  Type (or paste from virtual clipboard) to enter text in the box as necessary to describe the page.

Page Number

Required.  Enter the numeral used to mark the relevant sequential page within the form template.

Sponsor Code

Required.  Type the value if known in the box, otherwise, search for and select it and populate the box automatically by using either the lookup or direct inquiry functions.

Attachment

Browse for and attach the form template document using your computer’s File Upload functionality (or operating system equivalent).
