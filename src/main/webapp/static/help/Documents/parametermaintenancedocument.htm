Parameter Maintenance Document

The Parameter Maintenance Document is used to define parameters and business rules in the KC. A specific value of a parameter can vary based on what the parameter is used to define. These rules create restrictions and enforce valid values and combinations on various document types or batch processes. The value may, for example, be text that KC is to display in a given location or it may be a simple yes or no value to turn an option on or off.
