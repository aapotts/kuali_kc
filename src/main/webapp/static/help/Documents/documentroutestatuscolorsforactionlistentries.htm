Document Route Status Colors for Action List Entries

The Document Route Status Colors for Action List Entries section of the Workflow Preferences screen allows you to selectively color Action Requests based on the Route Status.  Click one of the color options for each document route status. The row containing the document of the status is displayed in the color of your choice in the Action List.
