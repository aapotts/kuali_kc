Object Code Name sub-panel

Click  to expand the sub-panel.

The top section of this sub-panel lists all personnel associated with the Object Code Name identified in the title of the sub-panel.  Data entered on this panel applies to the individual person or role listed on the same line.

Table 77 Object Code Name fields

Field

Description

Person

The name you selected on the Person drop-down list and associated with an Object Code Name now appears on the sub-panel with that Object Code Name. 

Start Date

 

Required

 

First day on which this individual's salary is a part of the project budget

End Date

Required

 

Last date on which this individual's salary is a part of the project budget

% Effort

Optional

 

Defines that portion of the individual's salaried time to apply to the budget

If this field is empty, KC returns 0.00 for calculated salaries.

 

If your proposal sponsor is the NIH, Person Months is an NIH term used in place of % Effort.

% Charged

Optional

 

Defines that portion of the individual's salaried time requested from the sponsor and which KC uses when performing calculations involving rates 

See Rates tab.  See Rate Classes sub-panel.  If this field is empty, KC returns 0.00 for Rate Cost and Rate Cost Sharing.

 

The value entered in the Percent Charged field cannot exceed the value entered in the Percent Effort field.

Period Type

Required

 

Click  to view a drop–down list of Period Types.

Highlight and select the correct Period Type.

KC defaults to Calendar. 

Your institution sets the items displayed in this list. 

Standard options for this list are Academic calendar year (Sept. thru August), Calendar year (Jan. thru Dec.), and Summer period.

 

The Period Types (Academic, Calendar, Summer) are fixed, and do not drive salary calculations. 

Period type aids in converting % Effort into person–months for proposals submitted to federal agencies.

Grants.gov requires person months to express effort.

Requested Salary

Display Only

 

Shows the total salary requested for the individual or role

Requested Salary figures come from Base Salary times % Effort charged to the project. 

Calculated Fringe

Display Only

 

Calculated Fringe is a percentage of the individual's requested salary calculated by KC.  It is affected by fringe rates on the Rates tab.

Click  to remove the person's name and details from the sub-panel.

This action does not delete the individual from the project.
