Edit OrganizationTypeList Panel

The Edit OrganizationTypeList panel lets you create a new Organization Type Code.  You must enter a code (abbreviation) for the new Organization Type Code and a description for it.

 

Figure 580 Edit OrganizationTypeList Panel

 

Table 271  Edit OrganizationTypeList Panel Fields

Field

Definition

Organization Type Code

Required

Enter a number for this Organization Type Code.  The type code must not be the same as an existing Organization Type.

Description

Required

Enter a descriptive name for this Organization Type Code.

 

 

 

Edit existing values only if the new description is synonymous with the original term.  You can add new values. You should not delete existing values.
