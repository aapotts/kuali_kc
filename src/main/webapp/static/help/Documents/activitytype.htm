Activity Type

The Activity Type Maintenance Document allows you to specify the list of activities from which to select to specify the type of activity proposed, (for example, organized research or instruction) in Proposal documents.  Activity type refers to the major functions of an institution based on OMB Circular A-21.  Per OMB Circular A-21, the major functions of a "university" are instruction, sponsored instruction and teaching, departmental research, organized research, sponsored research, university research, other sponsored activities, and other institutional activities.  Each account in an institution's financial system must be identified by its activity type.
