Edit Frequency Base Panel

The Edit Frequency Base panel lets you create a new Frequency Base Code.  You must enter a code (abbreviation) for the new Frequency Base Code and its Description.  Both must be unique:

 

Figure 528 Edit Frequency Base Panel

 

Table 224  Edit Frequency Base Panel Fields

Field

Definition

Frequency Base Code

Required

Enter a value to identify this Frequency Base Code.  You may use one or more letters and/or numbers.

Description

Required

Enter a descriptive name for this Frequency Base Code.

 

For information about creating a new maintenance document, see Initiating A Maintenance Document in Maintenance Document Operations.

For information about routing a maintenance document, see Submitting A Maintenance Document in Maintenance Document Operations.
