Sort Column

Sort column is a toggle button that allows you to sort the display of a search results table by the items in that column.  The Sort Column function may also appear as underlined text in a column header.

 and

In both cases, the action toggles the sort order.  The first sort is from smallest to largest.  The second sort is from largest to smallest.  Clicking  or the underlined header text once sorts the column from 1–9 or A–Z and clicking it again reverses the sort so that it is from 9–1 or Z–A.

KC sorts numbers separately on the first digit, then on the second digit.  Therefore, KC starts a list with 1, followed by 10, then 11, then 12, and so on (see Figure 52).  This may cause short lists to appear out of sequence, but it is necessary when sorting columns of numbers with two or more digits:
