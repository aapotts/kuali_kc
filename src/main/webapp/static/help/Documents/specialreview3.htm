Special Review

The Special Review page of the Protocol document is configurable by implementing institution to meet unique review requirements.  It provides functionality that allows you to select and enter information that you can then add as individual numbered rows in a table below the selection/addition row and save as part of the Protocol document.  Special Review is shared by other modules in KC including Award and Proposal.

Use the drop-down menus by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Click the calendar icons to select, or type in mm/dd/yyyy format to populate the box with the desired date.

Click the add  button to add your selection to as a row in the table below, which will become a numbered line item. 

Click the delete button to remove a previously-added line item row from the table.

  For more information about basic line item functionality, see Common Line Item Operations in Common E-Doc Operations.

Click the save button at the bottom of the page to save the line items you have added.
