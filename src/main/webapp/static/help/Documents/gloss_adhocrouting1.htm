Ad Hoc Routing

A type of routing used to route a document to users or workgroups that are not in the Routing for the given Document Type. After the Ad hoc Routing is completed, the designated routing returns to its prescribed path.  Phrase. To route a document to a user or role that normally would not receive it. The routing returns to the prescribed path after the Ad Hoc Routing is complete. Also the route level in which most transactions begin their routing.
