Edit Sponsor Template Reports Panel

The Edit Sponsor Template Reports panel lets you create one or more New Sponsor Template Reports.  For each New Sponsor Template Report, you must select a:

•      Report Class Code

•      Report Type Code

•      Frequency Code

•      Frequency Base Code

•      Osp File Copy Code

You can use Sponsor Template Reports show report requirements of a sponsor.  When synchronized with an award, the Sponsor Template updates the award with the report requirements of this sponsor.  These reports generate outside of KC.

Click  to create another New Sponsor Template Report section.
