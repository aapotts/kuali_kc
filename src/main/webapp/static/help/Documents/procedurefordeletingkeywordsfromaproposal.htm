Procedure for Deleting Keywords from a Proposal

Deleting a keyword disassociates it from that proposal.  If you disassociate it, then you cannot find that proposal using that keyword when you do a look up or create a report by keyword. 

When you delete a keyword, KC keeps that keyword in the keyword list.  You can re-associate this keyword with this proposal or associate it with other proposals using the Adding Keywords to a Proposal procedure above.
