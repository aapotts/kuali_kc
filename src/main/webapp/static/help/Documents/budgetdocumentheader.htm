The document header on a Budget Document is slightly different from the document header on other e-docs.  Most document headers display six fields of identifying information.  Yellow and blue highlights in

The Budget Document header displays the Budget Name and budget Version #, but does not show the Sponsor Name or the PI.

The Award Document header is also different from most other document headers.  (See Award Document)

 

Table 7 Budget Document Header Field Descriptions

Field

Description

Document Nbr

The Budget Document and the Proposal Development Document share this document number.

Initiator

The user name or login ID of the person who first created this budget version

Budget Name

User-defined name for this budget version

Status

The current status of the Proposal Development Document to which this Budget Document is attached

This is the completion status. 

 

This is not the routing or submission status.

Created

Date-time stamp

This shows you when someone created the Proposal Development Document for this Budget Document.

 

This is not the creation time and date for this budget document or this budget version.

Version #

The system-assigned unique budget version number
