Successful XML Extraction from a PDF File

Upon successful XML extraction, KC adds a line item to the Sub Award Budget panel. 

 

Figure 270 Sub Award Budget Panel Following a Successful XML Extraction

 

Figure 271 Details Sub–panel on the Sub Award Panel

 

The Sub Award Budget panel shows one line item for each successful XML extraction. 

Each line item shows the Organization Name, any Comments entered, and the Sub Award PDF File Name.

Each line item has one Details sub–panel.
