Edit Intellectual Property Review Result Type Tab

The Edit Intellectual Property Review Result Type panel lets you create a new Intellectual Property Review Result Type Code.  You must enter a code (abbreviation) for the new Intellectual Property Review Result Type Code and a description for it.

The IP Review Result Type maintenance document has a unique Edit Intellectual Property Review Result Type panel in addition to the standard panels for a maintenance document.

 

Figure 540 Edit Intellectual Property Review Result Type Panel

Table 238  Edit Intellectual Property Review Result Type Panel Fields

Field

Definition

Intellectual Property Review Result Type Code

Required

Enter a value to identify this Intellectual Property Review Result Type Code.  You may use one or more letters and/or numbers.

Description

Required

Enter a description for this Intellectual Property Review Result Type Code.

Current KC descriptions for Intellectual Property Review Results Codes are:

A:  No inventions disclosed to the Technology Transfer/Technology Licensing Office

B1:  Inventions disclosed, but not related to this proposal

B2:  Inventions disclosed that relate to this proposal, but not available for this project

B2-R:  Inventions disclosed that relate to this proposal, and are available for this project

 

 

You may modify any of these existing values.
