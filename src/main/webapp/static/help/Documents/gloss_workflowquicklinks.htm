Workflow QuickLinks

A web interface which provides quick navigation to various functions in KEW. These include

Quick E-doc Watch Keep track of 5 most recent actions taken by user and available for user to select and repeat the action.

Quick E-doc Search Keep track of 5 most recent E-doc that are searched by user and available for user to select and repeat the search.

Quick Action List Keep track of 5 most recent document types the user had taken action with. The 5 document types are available for user to select and repeat the action.

Quick E-doc initiate Keep track of 5 most recent templates used by user and available for user to select and repeat the action.
