Summary

The Summary panel in the Time& Money tab contains read-only fields with data provided from other panels or sources. 

Fields marked with an * contain required information.  The Award ID, Sponsor ID, Award Status, Award Title, Award Type, Activity Type, Sponsor ID, and Project End Date fields always display data because they are required fields. 

Other fields in the panel only display data if your institution has entered it for this Award.
