Document Search (doc search)

KC provides a document search engine that you can use to locate and open your KC documents.  You use doc search to locate a specific document or a list of documents from which you can open a document to see its details.  You enter search criteria to create a list of specific documents or you can search without providing any search criteria to receive all documents. 

 

You must have view or modify rights to the specific document in order to open it.

For basic information about how to use the doc search feature, see  E-Doc Fundamentals.
