F & A and EB Default Rates

When KC looks for a default rate, it looks for rates in tables for the given year.  With F & A and EB rates, if your institution did not identify a default rate for the given year, KC uses the rate from the last year for which it finds a rate.  For example, say your institution entered 45% for an EB rate in FY07.  If you do not define a rate for FY10, KC looks at FY09 and if it does not find a rate, it goes to FY08.  KC continues searching until it finds the most recent rate entered for that unit.  In this case, the last rate entered was 45% in FY07.  KC continues to use that rate until you update a more recent year.
