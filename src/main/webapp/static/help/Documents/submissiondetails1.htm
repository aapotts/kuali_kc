Submission Details

The Submission Details subsection of the Summary & History section includes five subsections:  Submission Details, Reviewers, Vote Summary, Check List Items, and Review Comments.  They allow you to view information pertaining to the review of the Protocol document.  The table below summarizes the purpose of each.

Figure 407 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection Layout

Submission Details

Includes the most general information about the Protocol that has been submitted to the committee.  It provides enough detail to allow for ease of identification of the submission.

Figure 408 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Submission Details Example

Reviewers

The PI is not able to view the Reviewer names and types, however, IRB Administrators are able to view them.

Figure 409 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Reviewers Example

Vote Summary

Allows you to view the number of votes by type.  Click the green arrow icon to view comments.

Figure 410 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Vote Summary Example

Check List Items

When pending review work to be accomplished by the committee is identified and entered, it appears here as numbered line items.

Figure 411 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Check List Items Example

Review Comments

Predefined comments selected by the committee members are displayed.  Private indicates the view is restricted to members only.  Final indicates if appropriate action is taken by the PI as a result of reading the comment, no further comment or revision of the comment will occur.

Figure 412 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Reviewer Comments Example
