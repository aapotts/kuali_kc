Proposal Type

Use the Proposal Type maintenance document to lookup, create and update Proposal Types.  Proposal types are defined, in part, by the Federal Government for Grants.gov submission. Proposal Type is a required field for saving an initial Proposal Development e-doc.

 

To access the Proposal Type Lookup screen:

Maintenance main menu tab > Shared menu group > Proposal Type link

Some of these items, like Proposal Type, map to Grants.gov submissions, so you should edit with caution.
