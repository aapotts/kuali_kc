Organization/Location Panel

The Organization/Location panel is where you will find the Applicant University’s address associated with the proposal.  Additionally, the name and address of the Performing Organization, Performance Site Locations (including sub contactor sites), and Other Organizations associated with the proposal are listed on this tab.  Figure 60 indicates where the Organization/Location panel is found and Figure 61 shows the primary information headings seen when the tab is opened.  Descriptions of the fields and sections found on the Organization/Location tab are found in Table 35.

Table 35 Organization/Location Panel Field Definitions and Procedures

Field

Description

Applicant Organization

 

The institution / organization that submits the proposal and will assume legal and financial responsibility for the award if granted.

Applicant Organization
Name field

The first time a new proposal is saved, the Applicant Organization name field is automatically populated with the name of the institution found at the top of the list of institutions available for populating various name fields in the proposal.  This default name should be the institution for which the user is performing the work of creating a new proposal.  The Applicant Organization field is read-only.  It cannot be edited.

Applicant Organization Address

Appears in a read only area to the right of the Applicant Organization name.

Applicant Organization congressional district

U.S.  congressional district information corresponding to the physical address of the Applicant Organization (e.g.  SD-002).  In the event an organization is in more than one congressional district, additional districts may be added by completing the empty State and District Number fields, then saving the new entry by clicking the Add button  located on the far right.

Performing Organization

The organization / institution at which project work will take place.

Performing Organization Name field

The first time a new proposal is saved, the Performing Organization field is automatically populated with the same organizational name placed by default into the Applicant Organization field. 

Performing Organization Address

Appears in a read only area to the right of the Performing Organization field.  In contrast to the Applicant Organization field, the Performing Organization Address field IS NOT read only and may be edited by first using the Clear  function on the far right of the Performing Organization section.  This will delete the information entered in the Name field and the Address area.  Use the Search  or Direct Inquiry  functions to repopulate the fields. 

If the proposal is saved while the Performing Organization fields are empty, the fields will repopulate with the default data when the proposal next appears on screen.  Only one Performing Organization may be identified in this section.

Performing Organization congressional district

U.S.  congressional district information corresponding to the physical address of the Performing Organization.  More than one congressional district may be listed for the performing organization.

For details, refer to - Applicant Organization congressional district.

Performance Site Locations

Performance Site Locations refer to the place(s) where proposed work will be performed, including subcontractor sites.  More than one Performance Site may be added to a proposal.  Each site must be entered separately. 

Location Name field

The left-most form field is for quick reference.  The name entered in this field (to the left of the Performance Site address) may be a project site designation, site nickname, or other reference including the formal name that appears with the address to the right.  This field is NOT updated when a Performance Site Address is selected using Search or Direct Inquiry..

Location Address

A Performance Site Address is added to the proposal by using the Search  or Direct Inquiry  functions.  The address of the Performance Site will appear with the formal name of the site or organization as with a mailing address.  Selecting an address will NOT update the Performance Site name entered on the left.

Performance Site congressional district

U.S.  congressional district information corresponding to the physical address of the Performance Site listed.  More than one congressional district may be listed for a single location.

For details, refer to - Applicant Organization congressional district.

Other Organizations

Other Organization Name field

The left-most form field is for quick reference.  The name entered in this field (to the left of the Other Organization address) may be a project site designation, site nickname, or other reference including the formal name that appears with the address to the right.  This field is NOT updated when an Other Organization Address is selected using Search or Direct Inquiry.

Other Organization Address

A Performance Site Address is added to the proposal by using the Search  or Direct Inquiry  functions.  The address of the Other Organization will appear with the formal name of the site or organization as with a mailing address.  Selecting an address will NOT update the Other Organization name entered on the left.

Other Organization congressional district

U.S.  congressional district information corresponding to the physical address of the Other Organization listed.  More than one congressional district may be listed for a single location.

For details, refer to - Applicant Organization congressional district.

Actions

to add a new Performance Site or Other Organization after entering its name and address

to clear the name and address of the Performing Organization

to remove an existing Performance Site or Other Organization
