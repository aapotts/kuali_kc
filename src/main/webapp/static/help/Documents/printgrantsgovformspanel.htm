Print Grants.gov Forms Panel

The Print Grants.gov Forms panel lists the proposal’s Grants.gov forms and options to print them.

 

Figure 197 Print Grants.gov Panel

 

You cannot print Grants.gov forms until you correct all errors on the Data Validation and the Grants.gov Errors panels.

 

 

Table 61 Print Grants.gov Forms Panel Definitions

Field

Description

Print Grants.gov Forms

Read only

The name of the Grants.gov form

Print Form Checkbox

Optional

The indication to print the form

Click  to select this form to print. 

Click  again to clear the selection so the form does not print. 

Actions

Click  to select all forms in the list for printing.

Click  to clear all forms from the selection list.

Click  to print the selected (check marked) forms.
