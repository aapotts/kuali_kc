Create a New Budget Version

On the Budget Versions panel, enter a name for your new budget in the Name field in the top row.  It does not need to be a unique name.  Click  in the Actions column.  KC registers the new budget and gives it a version number.  The version number is a unique identifier for this budget version among all the budgets attached to the proposal.  The version name is not the unique identifier.  Sometimes you may need to create several versions with the same budget name.

Select the budget you want to work with and click  in the Actions column on that row.

 

The Proposal Development Document and the Budget Document both use the Budget Versions tab.

If you select the Budget Versions tab while in the Proposal Development Document, you remain in the Proposal Development Document. 

When you open a budget version from the Budget Versions tab, the on-screen document changes from Proposal Development Document to Budget Document and you see details of this budget version on the Parameters tab on the Budget Document.  The document header reflects this change in documents.
