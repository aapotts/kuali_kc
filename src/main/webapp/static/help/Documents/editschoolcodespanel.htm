Edit School Codes Panel

The Edit School Codes panel lets you create a new School Code.  You must enter a code (abbreviation) for the new School Code and a description for it.

 

Figure 659 Edit School Codes Panel

 

Table 3 Edit School Codes Panel Fields

Field

Definition

School Code

Required

Enter a value to identify this School Code.  You may use one or more letters and/or numbers.

Description

Required

Enter a descriptive name for this School Code.

 

For information about creating a new maintenance document, see Initiating A Maintenance Document in Maintenance Document Operations.

For information about routing a maintenance document, see Submitting A Maintenance Document in Maintenance Document Operations.
