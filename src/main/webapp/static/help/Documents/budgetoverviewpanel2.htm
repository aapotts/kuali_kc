Budget Overview Panel

The Budget Overview panel displays a quick summary budget figures for the budget period shown in the panel header. 

Figure 251  Budget Overview Panel – Display Only Field Data Examples

 

Budget figures on this panel come from the Parameters tab, the Personnel tab, and the Non–Personnel tab.

 

Figure 252 Budget Overview Panel with Error Message

If total costs for a budget period or budget version exceed limits specified on the Parameters tab, an error message, "The Total Cost Limit has been exceeded", appears just below the Budget Overview section header.
