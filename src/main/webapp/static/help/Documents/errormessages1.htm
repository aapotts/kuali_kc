Error Messages

 

Figure 268 Sub Award Budget Panel Showing Error Message

The PDF you upload must contain form fields for the XML extractor to read.  If the  tool cannot locate data in form fields, an error message appears below the Sub Award Budget panel header.

 

Figure 269 Sub Award Budget Panel Upload File Size Error Message

 

If you attempt to upload a file larger than 5 megabytes, you will receive an error message below the Sub Award Budget panel. (See Figure 269)
