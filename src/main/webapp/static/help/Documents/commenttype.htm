Comment Type

The Comment Type maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.  For example, Comment Types are used in Award Templates and the Award e-doc to categorize comments saved with the template or award.  For instance, different comment types exist for Reporting, Property, and Procurement.

It is recommended that you not delete existing values, and further, that you only edit them if the new description is synonymous with the original term.

Edit Comment Type Section

The Edit Comment Type section of the Comment Type maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 423 Comment Type Maintenance Document - Edit Comment Type Section Field Descriptions

Field

Description

Comment Type Code

Required.  Enter the three-digit alphanumeric code value used to identify and reference this comment type.

Description

Required.  Enter a name that uniquely describes this comment type. Click the add note  icon to view/edit/paste text in a new browser window, then click the continue button to return to the text entry field in the document.

Template Flag

Optional. Click the check box if you want this comment type to appear in Award Templates. Click within the checkbox to select the item.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

Checklist Flag

Optional. Click the check box if you want this comment type to appear in the Checklist.  Click within the checkbox to select the item.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.

Award Comment Screen Flag

Optional.  Click the check box if you want this comment type to appear on the Award Comments, Notes & Attachments screen. Click within the checkbox to select the item.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.
