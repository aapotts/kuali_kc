Custom Data Tab

The Custom Data panel is used to display the Custom Attributes defined by each institution to collect data not captured on other Proposal and Budget document tabs.  Fields for inputting target attributes are displayed on panels, the titles of which are specified when the custom attributes are created. 

In the event your institution requires additional attributes for reporting purposes, they can be added as custom text fields or have associated reference tables to supply valid values.  Your technical staff may configure the data dictionary to make the field required or optional, make them available as search criteria and/or appear in search results of lookups, and include security and encryption as necessary.
