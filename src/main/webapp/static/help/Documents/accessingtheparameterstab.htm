Accessing the Parameters Tab

The Parameters tab appears in the Budget document.

You may also access the Parameters tab from two additional locations; both are on the Budget Versions tab:

1.  If you are in a Proposal Development document, you cannot see the Parameters tab.  Go to the Budget Versions tab on the Proposal Development document and open a specific budget version there.  This action takes you from the Proposal Development document to the Parameters tab on the Budget document.  The budget version # shows in the Budget document header at the top-right of your KC screen.

2.  If you are on the Budget Versions tab on the Budget document, you can see the Parameters tab and you can see the Budget Name field and Version # field in the document header in the upper-right of your screen.
