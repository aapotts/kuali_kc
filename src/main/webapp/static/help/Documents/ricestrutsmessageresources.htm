rice.struts.message.resources

Purpose:  Message bundle names to load.

Default Value:  ${core.rice.struts.message.resources}, ${kc.struts.message.resources}

Notes:  In the rice parameter rice.struts.message.resources messages in the bundles on the right will override the sames messages in bundles on the left. Given the default values, if a property [display.name=the dsiplay name] exists in the org.kuali.rice.ksb.messaging.ApplicationResources and ApplicationResources (kc's message bundle) kc's message will take precendant.
