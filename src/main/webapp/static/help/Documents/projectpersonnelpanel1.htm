Project Personnel Panel

This panel displays all project personnel.  Changes to personnel made on the Key Personnel tab show up in the Project Personnel panel when you click .

Figure 239 Project Personnel Panel

 

The Project Personnel panel displays (All Periods) in the panel header, regardless of which budget period you select for display in the other panels.

Figure 240  Add Project Personnel Selection Area

 

Add additional personnel using the search operations displayed in the top row in the Person column.  Changes in project personnel do not show in the Personnel Details panel until saved.
