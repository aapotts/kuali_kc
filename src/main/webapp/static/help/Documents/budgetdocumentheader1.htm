Table 65 KC Budget Document Header Field Descriptions

Field

Description

Doc Nbr

The Budget Document and its Proposal Development Document have the same document number.

Initiator

The user name or login ID of the person who created this budget version

Budget Name

Your institution’s name for this budget version

Status

Status of the Proposal Development Document associated with the Budget Document

The status shows document completion. 

This is not the routing or submission status.

Created

The time and date the Proposal Development Document was created

This is not the creation time and date for either the Budget Document or the Budget Version.

Version #

The system-assigned, unique budget version number
