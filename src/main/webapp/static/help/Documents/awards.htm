Awards

The Awards group on the Resercher menu provides links to Lookup screens that allow you to search for existing Award documents and view summarized information about them.  From the search results, links are provided that allow you to drill down into the documents themselves, or view additional details about their content.

All my Awards

Takes you to the Award Lookup screen, which displays tools for entry and selection of various award-related criteria that allow you to refine and conduct a search for Award documents.  From the search result table, you are then able to open, copy or view medusa information about each Award document.

  For general information about using the Document Search screen, see Searching for a Document in Common E-Doc Operations.
