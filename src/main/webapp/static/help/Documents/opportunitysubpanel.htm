Click  to select a Submission Type:

 

If you leave the Submission Type as Selected when you submit this proposal, you receive an error during data validation.  To fix that error, select a Submission Type of Application, PreApplication, or Changed/Corrected Application.

Conditionally required: This field is required only if the S2S Revision Type is Other.

The description of the other revision type

Enter the Description into the box.

 

If you select an S2S Revision Type of Other and you do not enter a Description, you receive this error when saving the Grants.gov tab:

Enter the Description or select another S2S Revision Type to correct this error.

 

If you enter a Description and you do not select an S2S Revision Type of Other, you receive this error when saving the Grants.gov tab:

Select a Revision Type of Other or clear the Description box to correct this error.

CFDA Number

 

Read-only

Unique identifier for the sponsor and the funding opportunity announcement (FOA)

The first 2 digits identify the Federal agency.  The last 3 digits identify the specific Federal program.

Competition Id

 

Read-only

Opening Date

 

Read-only

Closing Date

Instruction Page

Read-only

The URL (web address) of the instructions for this opportunity, provided by the sponsor at www.grants.gov

Use this address to download the instructions.

Schema URL

 

Read-only
