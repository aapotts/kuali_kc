You Are Here: Kuali Enterprise Workflow (KEW)

Action List   

When a user submits a document to the KEW workflow, KC routes an action request to the appropriate individuals or workgroups.  At the same time, KC adds the document and its action request to each approver’s Action List.  Your Action List contains all of the documents that are pending your action.  From your Action List you will open each e-doc to review and approve.

 

You can change your Action List display by setting your Preferences.  For more information about Preferences, see  Kuali Enterprise Workflow, Preferences.

You can display sets of documents by setting up a Filter.  For more information about Filter, see  Kuali Enterprise Workflow, Filter.

For information about how to approve a document using Workflow Approval Buttons, see  Kuali Enterprise Workflow, Workflow Action Buttons
