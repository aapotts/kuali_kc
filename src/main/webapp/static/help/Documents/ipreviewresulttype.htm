IP Review Result Type

The IP Review Result Type maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.  The Technology Transfer Office (TTO) at your institution reviews proposals with commercial entities for background intellectual property.  The TTO records the results of their review by selecting a review type in KC.  The IP Review Result Type Code maps to a specific review status:  For example, the code A might mean No Inventions Disclosed, and the code B1 might mean Inventions Disclosed, Not Related to Proposal.

Edit Intellectual Property Review Result Type Section

The Edit Intellectual Property Review Result Type section of the IP Review Result Type maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 543 IP Review Result Type Maintenance Document - Edit Intellectual Property Review Result Type Section Field Descriptions

Field

Description

Intellectual Property Review Result Type Code

Required.  Enter a value to identify this Intellectual Property Review Result Type Code.  You may use one or more letters and/or numbers.

Description

Required.  Enter a description for this Intellectual Property Review Result Type Code.  Current KC descriptions for Intellectual Property Review Results Codes are:

•      A:  No inventions disclosed to the Technology Transfer/Technology Licensing Office

•      B1:  Inventions disclosed, but not related to this proposal

•      B2:  Inventions disclosed that relate to this proposal, but not available for this project

•      B2-R:  Inventions disclosed that relate to this proposal, and are available for this project
