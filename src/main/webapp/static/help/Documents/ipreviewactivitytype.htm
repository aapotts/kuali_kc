IP Review Activity Type

Use the IP Review Activity Type screen to lookup, create, and update Intellectual Property (IP) Review Activity Type Codes.  Technology Transfer Offices use IP Review Activity Types to record an activity while reviewing a proposal for Intellectual Property issues.

For example, your Technology Transfer Office may use Activity Type Codes to record when a staff member attends a meeting, sends an email or fax, or calls another person on an IP Review.

Subtopics:

On the Intellectual Property Review tab of the KC Institutional Proposal e-doc, you select an Activity Type Code from the Activities panel.
