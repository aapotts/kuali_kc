Copy to New Document Panel

The Copy to New Document panel allows you to copy the current Proposal Development document to a new Proposal Development document.  On this panel, you can select options to include budget versions and attachments with the copy, and you can change the lead unit in the new Proposal Development document:

 

Figure 202 Copy to New Document Panel

 

You may use this panel to copy a Proposal Development document so you can change its Lead Unit.

 

 

After you save the Lead Unit on the Required Fields for Saving Document panel, you cannot change it.  You can only change a proposal’s Lead Unit  by making a copy of the proposal.
