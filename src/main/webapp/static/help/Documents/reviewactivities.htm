Review Activities

The Review Activities section contains information regarding specific IP review activities and dates.  There is one subsection for each review activity added. 

•      You can create a new review activity using the New Activities section. 

•      You can create a copy of an existing review activity using the New Copy column next to the Original column of the activity you want to copy.
