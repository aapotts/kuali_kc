Attachments Panel

The Attachments panel enables OSP administrative staff to attach files for reference about the project or award.

Click  on the Attachments panel to display its information:

 

Figure 101 Attachments Panel

 

Table 27 Attachments Panel Field Descriptions

Field

Description

Last Update:

Read Only

 

The date and time the note and/or attachment was added to the document

 

Updated By:

Read Only

 

The name of the person who added the attachment to the document

 

Attachment Type

Required

 

The type of attachment; your institution defines attachment types.  Examples of attachment types are Notice of Award and Award Modification.

 

Click  to select the Attachment Type.

Description

Optional

 

The description of the attachment

 

Expand the Description field for additional comments by clicking  to open a pop-up box.

 

Enter Description into the box.

File Name

Optional

 

Use this to locate a file on your local or shared drive and attach it to this document.

Click  and use your operating system’s standard procedure for locating and selecting the document you need to attach.

Actions

Optional

 

Click  to attach this file to this award.

For more information about adding an attachment, see  Common E-Doc Tabs and Panels, Notes and Attachments Panel, Procedure for Adding an Attachment.

 

Click  to open and view this attachment.

For more information about viewing an attachment, see  Common E-Doc Tabs and Panels, Notes and Attachments Panel, Procedure for Viewing an Attachment.

 

Click  to delete an attachment from this document.

For more information about deleting an attachment, see  Common E-Doc Tabs and Panels, Notes and Attachments Panel, Procedure for Deleting an Attachment.
