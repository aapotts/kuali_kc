Notes Panel

The Notes panel lets you write and save references about the project or award.

Restricted notes can only be created and seen by OSP staff.

Click  on the Notes panel to display its information:

Table 26 Notes Panel Field Descriptions

Field

Description

Posted Timestamp

The date this note was created

 

Author

The user who created this note

 

Note Topic

The title or label for the note

Enter the Note Topic into the box.

Note Text

Optional

 

The text or content of the note

 

Expand the Note Text field for additional comments by clicking  to open a pop-up box.

Enter the Note Text into the box.

Restricted?

Optional

 

Indicates whether this note is restricted to OSP Administrators

 

Restricted notes do not print or display on the award notice or modifications.  Only people with permission to maintain the notepad can view or update restricted notes.

Click to restrict access to the note.
