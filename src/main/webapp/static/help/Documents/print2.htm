Print

The Print section of the Protocol Actions page allows you to select from several printing options and click a button to execute a print command to your local printer.

Only one radio button option may be selected for printing at a time.

Figure 403Protocol Document, Protocol Actions Page – Print Section Example

 

Table 251Protocol Document, Protocol Actions Page – Print Section Field Descriptions

Option

Description

Summary View

Approximately one page of key data elements of the Protocol document

Full Protocol

The entire Protocol document in its current state

Protocol History

Summary view of the Protocol document at key points during its life cycle

Review Comments

Reviewer comments associated with the Protocol document

Attachments

Current versions of attachments associated with the Protocol document

To print:

•      Select the check box by clicking within the box to place a checkmark within it to indicate your selection of the desired section(s).

•      Click the print selected  button.

Depending on your browser type and its settings, a dialog box may appear that prompts you to open or save the document.
