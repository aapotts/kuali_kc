Sync to Template Function

The Sync to Template button copies a set of default comments stored in a Sponsor Template into this award.  Someone must add the sponsor template on the Award tab before you can sync comments from the template.  If a template isn’t already attached to the award when you click Sync to Template on the Comments panel, KC displays a warning message that you need to add a template. 

 

The Sync to Template button replaces comments already in the Comments, Notes and Attachments tab with the default comments from an attached template.

 

Click  to:

•      Enter information from the attached template into a new award.  Sync to Template automatically enters the existing template information into a new award.

•      Enter information from the attached template into the existing award on your screen.  This overwrites entries already in the fields of this award.  You can use Sync to Template even if some fields are empty in the existing award.

Your final step with the Sync to Template function is to confirm the synchronization.

 

Figure 97 Confirmation screen

 

If you click the yes button on the Confirmation screen, KC replaces all previous information in the Comments panel with the template comments.
