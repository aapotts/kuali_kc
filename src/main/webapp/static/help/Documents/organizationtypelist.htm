Organization Type List

Use the Organization Type List screen to look up, create, and update Organization Type Codes.  Organization Type refers to the applicant’s legal entity description.  Use Organization Type to classify the applicant institution as it relates to proposal submissions, particularly Grants.gov S2S (system-to-system) proposals.

The term organization refers to an institution that has implemented KC.  Organizations create and submit proposals to a variety of sponsors.  The term applicant has the same meaning as organization.  You may use applicant instead of organization.

Some examples of Organization Types include state government, local government, and institution of higher education (public or private).

 

Follow this path to see where Organization Type appears in the development of a proposal:

Researcher Main Menu Tab > Proposals menu group > Create Proposal link > Proposal Development Document e-doc > Proposal tab > Organization/Location panel > Applicant Organization and Performing Organization sections

In these Applicant Organization and Performing Organization sections, click  to search for the Applicant Organization and the Performing Organization.  Click  to view details about the organization, including its Organization Type.
