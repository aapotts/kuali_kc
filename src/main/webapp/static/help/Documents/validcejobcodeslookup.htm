Valid Ce Job Codes Lookup

When you click the Valid Cost Element Job Code link in the Shared menu group on the Maintenance main menu tab, KC displays the Valid Ce Job Codes Lookup screen:

 

Figure 716 Valid Ce Job Codes Lookup Screen

 

Use the Valid Ce Job Codes Lookup screen to search for and display existing valid Cost Element and Job Code combinations.

Enter appropriate information in the search fields to find the valid Cost Element /Job Code combination that you need (or leave all the fields blank to display all valid Cost Element/Job Code combinations), then click  to display a list of valid Cost Element/Job Code combinations.  When you click search, KC displays the Valid Ce Job Codes Lookup screen again, with the results of your search displayed below the buttons:

Figure 717 Valid Ce Job Codes Lookup Screen with Search Information

 

KC displays the valid Cost Element/Job Code combinations that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, you can either edit or copy a Cost Element/Job Code combination using the links in the Actions column.

For information about editing an existing maintenance document such as an IP Review Activity Type, see Editing A Maintenance Document in Maintenance Document Operations.

For information about copying a maintenance document such as an IP Review Activity Type, see Copying A Maintenance Document in Maintenance Document Operations.

The Valid Ce Job Codes Lookup screen has a  button in the top right corner that lets you create a new valid Cost Element/Job Code combination.

See the Valid Cost Element Job Code Screen: Panels section below for additional information about creating a new valid Cost Element/Job Code combination.
