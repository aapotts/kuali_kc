Route Log

The Route Log displays the workflow routing status of the document.  It includes document identification information, and completed, pending and future approval information.

 

Figure 423 Route Log

 

For more information about the Route Log fields, contact your KC System Administrator.

 

The Route Log contains four sub-panels:

 

Table 158 Route Log Sub-Panels Definitions

Field

Description

ID

Document identification and current workflow status

Actions Taken

Detailed information about completed workflow action requests; lists the status of actions by all persons and workgroups in previous workflow nodes

Pending Action Requests

Detailed information about the action requests in this workflow node; lists the status of action requests for persons and workgroups in this workflow node

Future Action Requests

Detailed information about the action requests for the next nodes in the workflow; lists the status of action request for persons and workgroups in the next workflow nodes
