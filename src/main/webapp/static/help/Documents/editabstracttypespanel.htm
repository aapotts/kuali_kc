Edit Abstract Types Panel

Unique to this maintenance document, this panel allows you to specify the details associated with the reference table, including a description of the abstract type.

Table 169 Edit Abstract Types Panel Fields

Field

Description

Abstract Type

Required

 

Enter the alphanumeric code value used to identify and reference this individual type of abstract. The code must be unique.

Description

Required

 

Enter a name for the abstract type that uniquely represents a summarized explanation of its meaning and/or purpose (for example, a title, several words separated by spaces, or a short phrase)

The abstract, or project summary, describes the project in a concise, clear, and brief way.  It should outline the problem, the objectives, expected outcomes, including significance of the project to the field of study.

Write the abstract in less technical language than the proposal narrative

 

 

Edit existing values only if the new description is synonymous with the original term.  You can add new values. You should not delete existing values.
