Range Operators

If you use range operators in a date search, all except double periods (..) must be before a date.  The double periods operator (..) should separate date values. 

 

Table 8 Range Operators Allowed in KC When Searching for Numbers and Dates

> 

Greater than

< 

Less than

>=

Greater than or equal

<=

Less than or equal

..

Through (for example: 12/03/2010..1/20/2011, which means from 12/03/2010 through 01/20/2011)
