Data Validation Panel

The Data Validation panel allows you to identify and correct errors and warnings in the proposal:

 

Figure 192 Data Validation Panel

 

KC automatically runs data validation when you submit your proposal to KC workflow routing. 

 

You may turn data validation on and off while you are working on your proposal. 

 

During the data validation process, KC checks all mandatory and conditional requirements for the proposal submission to KC workflow routing and to www.grants.gov.

 

KC flags missing or incorrect mandatory requirements as errors.  It flags missing or incorrect conditional requirements as warnings. 

 

KC notifies you about errors and warnings it finds on the Validation panel.  Each error and warning includes a description with a link to the area within the document so you may correct it. 

 

 

You can submit this proposal to the KC workflow routing when there are no errors in the Validation Errors panel.

You can submit this proposal to www.grants.gov when there are no errors in the Validation Errors and Grants.gov panels.

You can print PDF forms when there are no errors in the Validation Errors and Grants.gov panels.

Validation warnings do not prevent your submission of the proposal to KC workflow routing or www.grants.gov.
