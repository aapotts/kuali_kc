Account Type

Use the Activity Type Maintenance Document to lookup, create, and update Activity Types.  Activity type refers to the major functions of an institution based on OMB Circular A-21.  Per OMB Circular A-21, the major functions of a "university" are instruction, sponsored instruction and teaching, departmental research, organized research, sponsored research, university research, other sponsored activities, and other institutional activities.  Each account in an institution's financial system must be identified by its activity type.  On the Proposal tab of the KC Proposal Development e-doc, you select an Activity Type from the Required Fields for Saving Document panel.
