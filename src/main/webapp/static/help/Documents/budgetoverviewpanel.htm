Comments

On/Off Campus

 

Drop-down list where you can select Default, All On, or All Off, indicating where the work performed under this budget version is to take place and how F&A costs are calculated. 

The field shows Default for new budget versions. 

Default indicates that each line item in the budget includes a manual On / Off Campus decision. 

 

 

•    Select All On to indicate that all line items in this budget version are scheduled on-campus and should be calculated at the on-campus F&A rate.

•    Select All Off to indicate that all line items in this budget version are scheduled off-campus, and should be calculated at the off-campus F&A rate.

All On and All Off set the F&A rate for all line items in the budget.  If you select one of these, you do not have to enter any other On/Off Campus choices.

Modular Budget?

Optional

A check in the box  indicates that the funding requested is in the Modular Budget format (e.g.  $25,000 increments, up to a maximum of $250,000 per year)

Modular budgeting is used for proposals to the NIH.

Total Cost Limit

Optional

Manual entry field indicating the maximum cost (including indirect costs) paid by the sponsor.  If the budget exceeds the Total Cost Limit you enter here, a warning message appears on the screen. 

Residual Funds

Optional

Manual entry field indicating how much is remaining from a previous award.

F&A Rate Type
