Review Activities Panel

The Review Activities panel contains information regarding specific IP review activities and dates.  There is one panel for each review activity. 

•      You can create a new review activity using the New Activities panel. 

•      You can create a copy of an existing review activity using the New Copy panel next to the Original panel of the activity you want to copy.
