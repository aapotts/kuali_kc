Distribution (OSP File Copy)

The Distribution screen title will be changed to the OSP File Copy screen title.  The panel names, field names, and so forth that use the term Distribution will also change to use the name OSP File Copy.

Use the OSP File Copy screen to look up, create, and update OSP File Copy Codes.  The Office of Sponsored Projects (OSP) designates which documents you must forward to the OSP and which documents you must retain in the OSP award file for future use.  Documents retained for future use in the OSP award file include submitted invoices and/or financial reports sent to the sponsor.

For example, when the OSP sends a financial report to the sponsor, the OSP decides which documents to retain as a hard copy.  OSP may choose to retain a hard copy of the report, the report and letter, or some other combination.
