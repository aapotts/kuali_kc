Project Personnel Panel

The Project Personnel panel allows you to identify contacts associated with an award project.  You must select a Person and a Project Role for each person you add in the Project Personnel section.

When you click the Contacts tab, KC displays this screen:

 

Figure 78 Project Personnel Panel Screen on the Contacts Tab

 

Table 18 Project Personnel Panel Fields

Field

Definition

Person

Required

 

The name of an Employee or Non-Employee project person

Click the appropriate  to find and select an Employee or a Non-employee.

Unit

Read only

 

This person’s unit

When you add a person, KC automatically displays that person’s Unit.

Project Role

Required

 

The project person’s role on the project:

 

 

Click  to select the Project Role for this person.  If you select Key Person as the Project Role, you must also enter the individual’s specific role in the box.

Office Phone

Read only

 

This person’s office telephone number

When you add a person, KC automatically displays that person’s office telephone number.

Email

Read only

 

This person’s email address

When you add a person, KC automatically displays that person’s email address.

Actions

Required

 

Click to add this person’s contact information to the Project Personnel panel.

Click  to delete the contact information for the person on this row from the Project Personnel panel.
