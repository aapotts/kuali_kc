Action List Controls

Action List controls include buttons to set up your workflow preferences and Action List filter, and to refresh your Action List.  There is also a link to your KEW Outbox:

 

Figure 412 Action List Controls

Click Outbox to go to a list of documents you have processed.

For more information about your KEW Outbox, see  Kuali Enterprise Workflow, Action List, Outbox.

Optional
