You Are Here: Kuali Enterprise Workflow (KEW) > Action List > Filter

Procedure for setting up a Filter:

 

1.  Click  to set up your Action List Filter:

 

    

     Figure 414 Filter Button

 

     KC displays the Action List Filter screen:

 

           

     Figure 415 Action List Filter Screen

 

2.  Set up your Action List Filter by entering information into the fields described in the table below. 

 

     Table 155 Action List Filter Criteria

Criteria

Description

Document Title

Optional

 

The full or partial document title you want KC to match for the filter

Enter the full or partial Document Title that you want KC to match; action requests that match this Document Title will be included in your Action List.

For example, when you enter “Test” in the Document Title field, the Action List displays all documents that contain the characters “Test” in the document description.

Click  to exclude this Document Title from the Action List.

The Document Title field is case sensitive.  You must enter upper and lower case letters exactly as you want KC to match them.

Document Route Status

Optional

 

The Document Route Status you want KC to match for the filter; action requests that match this Document Route Status will be included in your Action List.

Click  to select the Document Route Status.

For more information on Document Route Status, see  Kuali Enterprise Workflow, Route Status.

 

Click  to exclude this Document Route Status from the Action List.

Action Requested

Optional

 

The Action Requested you want KC to match for the filter; action requests that match this Action Requested will be included in your Action List.

Click  to select the Action Requested.

For more information on Action Requested, see  Kuali Enterprise Workflow, Ad Hoc Routing.

 

Click  to exclude this Action Requested from the Action List.

Action Requested Group

Optional

 

The name of the workgroup assigned to the action request; action requests that match this Action Requested Group will be included in your Action List.

Click  to select the Action Requested Group.

For more information on Action Requested Group, see  Kuali Enterprise Workflow, Workflow Example.

 

Click  to exclude this Action Requested Group from the Action List.

Document Type

Optional

 

The Document Type you want KC to match for the filter; action requests that match this Document Type will be included in your Action List.

Click  to search and select a Document Type.

Click  to exclude this Document Type from the Action List.

Date Created

Optional

 

The document creation date range; action requests that match this Date Created date range will be included in your Action List.

For example, you enter a Creation Date to of 01/01/2010, and a Creation Date from of 01/31/2010, for KC to match all documents created in January 2010. 

Creation Date to and Creation Date from are optional.

Enter the Date Created to in MM/DD/YYYY format, or click  to select the date.

Enter the Date Created from in MM/DD/YYYY format, or click  to select the date.

Click  to exclude this Date Created range from the Action List.

Date Last Assigned

Optional

 

The range of the dates the document was assigned to you; action requests that match this Date Last Assigned date range will be included in your Action List.

For example, you enter a Date Last Assigned to of 01/01/2010, and a Date Last Assigned from of 01/31/2010, for KC to match all documents created in January 2010. 

Date Last Assigned to and Date Last Assigned from are optional.

Enter the Date Last Assigned to in MM/DD/YYYY format, or click  to select the date.

Enter the Date Last Assigned from in MM/DD/YYYY format, or click  to select the date.

Click  to exclude this Date Last Assigned range from the Action List.

 

3.  Click  to save your Filter fields.  KC returns you to your Action List.  Your Action List displays only the documents that match your filter fields, unless you excluded them.  If you excluded them, your Action List displays documents that do not match your Filter fields.

 

When you click , KC displays an informational message indicating the fields your Action List is filtered to:

         

Figure 416 Example: Action List Filtered by Document Title

 

Click  or  to clear the information you entered in the Filter fields.  KC clears the information from all of the Filter fields.  You may reenter the information.

Click  to cancel the Filter and return to your Action List.
