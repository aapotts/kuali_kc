Summary Panel

The Summary panel has three sections: Personnel, Non-Personnel, and Totals.

 

Figure 235 Expanded Summary Panel on the Summary Tab of the Budget Document, with Sub-Panels Collapsed

 

Each section displays budget summary information for the budget version indicated in the document header.  Each section also displays sub-totals by budget period. 

Figure 236 Expanded Sub-Panels on the Personnel Section of the Summary Panel

 

When you collapse a sub-panel, it displays the sub-total for that budget category.  When you expand a sub-panel, it shows details for the line items in that budget category.  (See Figure 235, showing all sub-panels collapsed, and Figure 236, showing all the sub-panels for Personnel expanded)

 

Data will not appear in the Personnel section or the Totals section of the Summary panel on the Summary tab until someone enters it on the Personnel Detail panel of the Personnel tab. 

 

For more information, go to Personnel Detail Panel.

 

Data will not appear in the Non-Personnel section or the Totals section of the Summary panel on the Summary tab until someone enters it on one of the panels of the Non-Personnel tab. 

 

For more information, go to Non–Personnel Tab.

 

Table 71 More About the Summary Panel on the Summary Tab

Feature

Description

Personnel

Display-only list of personnel-related expenses by period

The Personnel section has three sub-panels that display personnel-related budget details and summaries. 

This data comes from the Personnel tab.

Non-Personnel

Display-only list of non-personnel expenses by period

The Non-Personnel section has sub-panels that correspond to the non-personnel object code names added as line items on the panels of the Non-Personnel tab.

Each new object code name that you add on the Non Personnel tab produces a new sub-panel in the Non-Personnel section of the Summary panel of the Summary tab.

KC takes the data entered for each object code name on the Non–Personnel tab and displays it on the corresponding sub-panel of the Non-Personnel section of the Summary tab. 

Totals

Display-only list of budget sub-totals and totals by category, period, and cost type for the current budget version

Navigation

When more than five budget periods exist, navigation buttons let you scroll left or right to view the additional periods.

Click  or  to move the display. 

When the budget version has five or fewer budget periods, these two navigation buttons are inactive (gray).

Action Buttons

Personnel section header

Click  to go to the Personnel tab, where you can enter or edit details about project personnel.

Non-Personnel section header

Click  to go to the Non-Personnel tab, where you can enter or edit details about project expenses not related to personnel.

 

After you enter data on the Personnel and Non-Personnel tabs, KC automatically saves the data when you go to another tab.  If your data does not show on the Summary tab, you may have lost your connection with the server while entering the data or while going to another tab. 

 

Best practices in data entry include regularly checking data display areas of your e-doc to confirm that your data is stored and retrieved properly.
