Award Type Lookup

When you click the Award Type link in the Awards menu group on the Maintenance main menu tab, KC displays the Award Type Lookup screen:

 

Figure 470 Award Type Lookup Screen

 

Use the Award Type Lookup screen to search for and display existing Award Type Codes.

Enter appropriate information in the search fields to find the Award Type Code you need (or leave all the fields blank to display all Award Type Codes), then click  to display a list of Award Type Codes.  When you click search, KC displays the Award Type Lookup screen again, with the results of your search displayed below the buttons:

 

Figure 471 Award Type Lookup Screen with Search Information

 

KC displays the Award Type Codes that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, you can either edit or copy an Award Type Code or Description using the links in the Actions column.

For information about editing an existing maintenance document such as an Award Type, see Editing A Maintenance Document in Maintenance Document Operations.

For information about copying a maintenance document such as an Award Type, see Copying A Maintenance Document in Maintenance Document Operations.

When your institution creates a new Award Type Code, it enters a Description for that code.

The Award Type Lookup screen has a  button in the top right corner that lets you create a new Award Type.

See the Award Type Screen: Panels section below for additional information about creating a new Award Type.
