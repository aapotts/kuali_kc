Question Category Lookup

When you click the Question Category link in the Shared menu group box on the Maintenance main menu tab, KC displays the Question Category Lookup screen:

 

Figure 611 Question Category Lookup Screen

 

Use the Question Category Lookup screen to search for and display existing Category Type Codes.

Enter appropriate information in the search fields to find the Category Type Code you need (or leave all the fields blank to display all Category Type Codes), then click  to display a list of Category Type Codes.  When you click search, KC displays the Question Category Lookup screen again, with the results of your search displayed below the buttons:

 

Figure 612 Question Category Lookup Screen with Search Information

 

KC displays the Category Type Codes that match your search below a statement about the number of items retrieved (how many items matched your search) and the number of items displayed.

In the search results list, you can either edit or copy a Category Type Code or Category Name using the links in the Actions column.

For information about editing an existing maintenance document such as a Question Category, see Editing A Maintenance Document in Maintenance Document Operations.

For information about copying a maintenance document such as a Question Category, see Copying A Maintenance Document in Maintenance Document Operations.

When your institution creates a new Category Type Code, it enters a Category Name for that code.

The Question Category Lookup screen has a  button in the top right corner that lets you create a new Category Type Code.

See the Question Category Screen: Panels section below for additional information about creating a new Category Type Code.
