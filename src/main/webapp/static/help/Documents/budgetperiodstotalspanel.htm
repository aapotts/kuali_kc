Budget Periods & Totals Panel

The Budget Periods & Totals panel displays all budget periods identified for this proposal.  Unless you add it using this panel, the data originates on the Budget Overview panel.

Figure 218 Budget Periods & Totals Panel

 

The end date of the first period displayed on the Budget Periods tab is automatically set to one year from the project start date.  If the project is longer than one year, subsequent periods are auto-generated in one-year increments.  The final period is less than one year if the project end date is less than 365 days from the start date of the last budget period.  The end date of the final period matches the project end date you enter on the Budget Overview tab.

The Project Start Date is the earliest date you may assign to a budget period.

The Project End Date is the latest date you may assign to a budget period.

Attempts to assign dates outside these two limits will result in an error message indicating the necessary correction(s).
