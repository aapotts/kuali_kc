Degree Type

Use the  Degree Type maintenance document to lookup, create, and update Degree Types.  Degree Types refer to degree(s) earned by key personnel listed in proposal records.  On the Key Personnel tab of the KC Proposal Development e-doc, you select a Degree Type from the Degree sub-panel for each listed person.
