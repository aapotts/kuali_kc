Unit Details Sub-Panel

The Unit Details sub-panel contains the units with which the Person is affiliated.

 

At least one Unit is required for the PI and for the Co-I. 

You cannot change the first Unit for the PI.  The first Unit for the PI must be the same as the Lead Unit for the proposal.  You may add or change other Units for the PI.

The Lead Unit for the proposal may be different from the PI’s Home Unit.  In this case, KC displays the Lead Unit in the second line of the PI’s Units.

KC automatically displays the Home Unit for the Co-I.  You may add or change Units for Co-Is.

KC initially displays this message for Key Persons:

Click  to display a Key Person’s Home Unit.

You may add or change a Key Person’s Unit.
