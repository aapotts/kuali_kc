Rule Lookup

The Rules link in the Workflow group causes the Rule Lookup screen to appear.  The Rule Lookup screen allows you to refine criteria to search for and view workflow rules.  The results allow you to view their associated templates and document types, and click “drill-down” links to view additional details for each.

Figure 112 Rule Lookup Screen – Layout and Search Results Example

 

For more information about general lookup screen functionality, see Lookup  in Selection, Entry & Action Tools.

 

 

Clicking the Rules menu link causes the Rule Lookup screen to appear.  The Rule Lookup allows you to search for routing rules using the standard Kuali search tools.

Figure 113  Rule Lookup Workflow Screen Search Criteria Section

From the search results table, you are able to click a link from the Rule Id column to access the Rule screen.

Figure 114  Rule Lookup Workflow Screen Search Results Table Example

The typical Rule screen includes four tabbed sections:

•      Rule

•      Person Responsibilities

•      Group Responsibilities

•      Role Responsibilities
