Non–Personnel Cost Category Panels

Use the remaining panels on the Non-Personnel tab for entering detailed costs to the budget.  Each panel header name identifies a broad cost category; the selected budget period is in parenthesis.  All of these panels have the same columns and fields.  The difference between panels is in the Object Code Name drop–down lists, which show object code names relevant to the cost category named in the panel header.

Each time you add a line item to any of these panels, KC adds to the panel a new Line Item Details sub-panel.
