Table 66 Budget Document Tab Descriptions

Tab

Description

Budget Versions

View summary information for each budget version in the current Budget Document. 

From this panel you can select individual budget versions to delete, modify, copy, or complete.

Parameters

View basic information about the Budget Document, including its status, relevant dates, costs, rates, and period totals. 

KC always displays this tab when you open the Budget Document from a Proposal Development Document.

Rates

Use this tab to override Institute Rates with Applicable Rates in this Budget Version.

Summary

View the expenses, calculated expenses, and total expenses in the Budget document.

Personnel

Search for and select employees involved in the proposed research so that their salaries and associated benefits become part of the budget.

Non-Personnel

Add non-personnel expenses to the budget, such as Travel, Equipment, and other operational, non-salary/wage expenses.

Distribution & Income

•      View cost sharing and unrecovered F&A distribution lists

•      add income amount lines by period

•      calculate summary totals

•      synchronize

Modular Budget

Create modular budgets.

View basic, direct cost, and F&A information about each budget.

Modify and synchronize that information.

Budget Actions

Select and print reports on budget content and activity

Record explanations to justify budget expenses

Add sub-award budget attachments for Grants.gov proposals
