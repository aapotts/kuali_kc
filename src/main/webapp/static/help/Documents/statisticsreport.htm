Statistics Report

The Statistics Report link displays a workflow screen that allows you to view the various statistical data about the operations of Workflow over a timeframe you are able to select.  It displays the counts of documents and users for various statuses and categories.

Figure 744 KC Statistics Report workflow screen

Modify the Begin and End Date fields as desired by entering in mm/dd/yyyy format or using the calendar  selector, select the desired option from the Average Number of Actions for Date Range list then click the refresh button.
