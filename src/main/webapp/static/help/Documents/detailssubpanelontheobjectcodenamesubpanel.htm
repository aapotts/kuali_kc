Optional

 

The number in this field should equal the number of persons listed on the Object Code Name sub-panel. 

For estimating purposes in advance of complete data entry, you may enter whatever number of personnel you anticipate for this Object Code Name.

Cost Sharing

Costs not billed to the project because a person's % Charged is less than their % Effort. 

Example: Bob's requested salary is $50,000 annually.  He spends 100% of his time on the project, but the institution bills only 75% of his salary against the project budget.  A source other than project funds must cover the other 25%.
