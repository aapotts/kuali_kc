Submission Details Sub-panel

The Submission Details sub-panel contains information about your proposal submission to www.grants.gov:

 

Figure 138 Submission Details Sub-Panel

 

 

KC displays this informational message on the Submission Details sub-panel before you submit the proposal to www.grants.gov:

KC automatically adds information to the Submission Details sub-panel after you submit the proposal to www.grants.gov.

 

Table 40 Submission Details Sub-panel Definition

Field

Description

Received Date

Read only

The date and time www.grants.gov received the proposal

Status

Read only

The current proposal status at www.grants.gov

GG Tracking Id

Read only

The tracking number assigned by www.grants.gov when you successfully submitted your proposal

Agency Tracking Id

 

Read only

The tracking number assigned by the sponsor, if applicable.

Not all sponsors assign an Agency Tracking ID.

Comments

Read only

Additional www.grants.gov information regarding the status of the application

Attachments

Read only

List of forms and attachments that you successfully submitted to www.grants.gov

Last Modified Date

Read only 

The date and time that KC last updated this application status from www.grants.gov
