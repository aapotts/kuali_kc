Correspondence Generated

The Correspondence Generated maintenance document allows you to select a combination of a Protocol Action Type and a Protocol Correspondence Type that together become a single “correspondence generated” record.  It is used to establish and maintain numeric code values that are stored in database tables.  This data serves as reference information that is then made available for display and selection in various areas of the system, including other e-docs.

Edit Correspondence Generated for Protocol Action Section

The Edit Correspondence Generated for Protocol Action section of the Correspondence Generated maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 629 Correspondence Generated Maintenance Document - Edit Correspondence Generated for Protocol Action Section Field Descriptions

Field

Description

Valid Proto Action Corresp ID

Required.  Enter a numeric value to uniquely identify this combination of Protocol Action Type and Protocol Correspondence Type in the system.

Protocol Action Type

Required.  Select an option from the list as desired.  Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Protocol Correspondence Type

Required.  Select an option from the list as desired.  Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Final Flag

Click within the checkbox to select the item to indicate the Correspondence Generated record is final.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.  When selected, the displayed value in the saved document appears as ‘Yes’.
