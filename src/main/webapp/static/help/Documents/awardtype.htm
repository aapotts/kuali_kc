Award Type

Use the Award Type screen to look up, create, and update Award Type Codes.  Award Type Codes identify the type of award instrument your institution receives from a sponsor. 

For example, Award Type Codes provide a way to distinguish between a Grant, a Contract, and a Cooperative Agreement.
