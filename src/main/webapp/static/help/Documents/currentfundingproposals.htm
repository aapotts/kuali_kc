Proposal Details

The Details sub-panel lists the details of the funding proposal and its budget.

When you click  on a Details sub-panel, KC displays information about that proposal:

 

Figure 468 Award Document, Award Page, Funding Proposals Section, Current Funding Proposals Subsection> Proposal Details View Mode Example Layout

 

Table 293 Award Document, Award Page, Funding Proposals Section, Current Funding Proposals Subsection> Proposal Details > Field Descriptions

Field

Description

The Institutional Proposal version number

Proposal Type

The type of submission, defined by your institution and/or the sponsor

Activity Type

 

The institution’s university function for this proposal, based on OMB Circular A-21

 

Per OMB Circular A-21, the major functions of a university are instruction, sponsored instruction and teaching, departmental research, organized research, sponsored research, university research, other sponsored activities, and other institutional activities.  Each account in an institution's financial system must be identified by its activity type.
