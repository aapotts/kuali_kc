You Are Here: Kuali Enterprise Workflow (KEW)

Route Status

You can find out each document’s workflow status on your Action List and in the Route Log:

 

Table 163 WF status

Status

Description

APPROVED

The document is approved. 

CANCELED

The document is denoted as void and should be disregarded.

COMMITTED

The document has been committed to the database. 

DISAPPROVED

The document is not approved.

ENROUTE

The document has pending approval requests.

EXCEPTION

The document has been routed to an exception queue because workflow has encountered a system error.

FINAL

There are no pending approval or acknowledgement requests.

INITIATED

The document is created but not yet saved or submitted to the workflow.

PROCESSED

The document has no pending approval requests but has one or more pending acknowledgement requests.

SAVED

The document is created and saved but not yet submitted to the workflow.
