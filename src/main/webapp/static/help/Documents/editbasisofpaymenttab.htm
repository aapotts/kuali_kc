Edit Basis of Payment Tab

Unique to this maintenance document, this tabbed section allows you to specify the details associated with the reference table, including a description of the basis of payment. 

Figure 478  Edit Basis of Payment Tab

 

Table 181  Edit Basis of Payment Tab components

Field

Description

Basis of Payment Code

Required.  Enter the numeric code value that is to be used to identify and reference this individual basis of payment.

Description

Required.  Enter a name for the payment basis that uniquely represents a summarized explanation of its meaning and/or purpose (for example, a title, several words separated by spaces, or a short phrase).

 

 

You may modify any of these existing values.
