Workflow Action Buttons

Workflow Action Buttons enable you to move the document through the KEW workflow routing.  Your institution uses workflow action buttons for the following processes:

•      Researchers and administrators submit documents for approval by department heads and administrators

•      Approvers, usually department heads and administrators, approve or disapprove documents

•      Administrators submit documents to www.grants.gov

 

KC displays Workflow Action Buttons based on:

•      your user role

•      the action requested of you

•      the document’s status in the KEW workflow

•      the document tab you are viewing

 

KC displays Workflow Action Buttons at the bottom of each document’s action tab:
