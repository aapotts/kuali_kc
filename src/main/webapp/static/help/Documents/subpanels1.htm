Sub-panels

Sub-panels are panels that you can display only if their parent panel is expanded.  If the parent panel is collapsed, click the  button next to the parent panel’s title to expand it.  Now you can display the sub-panel by clicking  next to its title.  Some sub-panels in KC don’t have a file tab graphic, but you still open them by clicking .
