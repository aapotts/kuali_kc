Route Log

The Route Log section of the Protocol Actions page displays four subsections of information about the document’s approval workflow.  These include ID, Actions Taken, Pending Action Requests, and Future Action Requests.  Each has show/hide buttons that allow you to display additional detail.  Some fields are display-only, while others display underlined information that, when clicked, display different screens containing even more detail in new browser windows.

Figure 416Protocol Document, Protocol Actions Page, Route Log Section Example

The Route Log section is standard for most e-docs in the KC system.

For more information about interpreting the displayed information and using the features of the Route Log, see Route Log  in Common E-Doc Tabbed Sections.
