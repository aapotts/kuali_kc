Business Rules

•      All users can look up table values

•      Only members of a predefined Initiators workgroup can initiate a document to create or modify the table values

•      Only members of a predefined table maintenance workgroup can approve maintenance documents

•      After approval, KC changes the status to FINAL and the reference table is updated with the new values
