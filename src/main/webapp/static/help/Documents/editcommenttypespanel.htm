Edit Comment Types Panel

Unique to this maintenance document, this tabbed section allows you to specify the details associated with the reference table, including a textual description of the comment type. 

 

Figure 491  Edit Comment Type Panel

Table 197  Edit Comment Tab components

Field

Description

Comment Type Code

Required.  Enter the three-digit alphanumeric code value used to identify and reference this comment type.

Description

Required.  Enter a name that uniquely describes this comment type.

Template Flag

Optional. Click the check box if you want this comment type to appear in Award Templates.

Checklist Flag

Optional. Click the check box if you want this comment type to appear in the Checklist.

Award Comment Screen Flag

Optional.  Click the check box if you want this comment type to appear on the Award Comments, Notes & Attachments screen.

 

 

 

Edit existing values only if the new description is synonymous with the original term.  You can add new values. You should not delete existing values.
