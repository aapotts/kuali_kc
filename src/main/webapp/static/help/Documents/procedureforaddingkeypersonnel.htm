Procedure for Adding Key Personnel

To add key personnel, you look up and select the person, assign their role, and add them to the proposal.  As you add each key person to the proposal, his or her Person Panel appears in the Key Personnel tab.  You may add both employees and non-employees to the proposal.

1.  Click Employee Search  to add an employee or click Non-Employee Search  to add a non-employee:

 

        Figure 145 Person Search

 

KC then displays either the KcPerson Lookup screen or the Non-Organizational Lookup screen. 

 

For more information on Lookup, see  E-Doc Fundamentals.

 

2.  Enter your lookup (search) criteria and return the person.

     

For more information on Lookup criteria, see  E-Doc Fundamentals.

 

After returning the person, KC displays the Add Key Personnel section.  The name of the person you select appears in the Person field:

 

      

         Figure 146 Selected Key Person

 

3.  Click  to select this person’s Proposal Role.

 

 

Figure 147 Select Proposal Role

 

You can only add one person with a Proposal Role of PI to a proposal.

You can only add a person once to a proposal. 

 

4.  Only when you select a Proposal Role of Key Person, KC displays a Key Person Role field.  Enter the Key Person Role in the box:

 

Figure 148 Add Key Person Role

 

     

 

5.  Click  to add this Person to the Key Personnel tab:

 

Figure 149 Add Personnel Button
