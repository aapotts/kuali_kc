Ynq Sub-Panel

The Ynq sub-panel displays the details about the question.  It lists the question’s ID number and status, valid answers, and whether a question explanation or date is required.  The question group is also listed:

 

Figure 178 Ynq Panel

 

Table 55 Ynq Panel Definitions

Field

Definition

Question Id

Read only

The system generated question identifier

Applies To

Read only

The KC module or document to which the question applies

Status

Read only

The question status

Question statuses are Active or Inactive:

•      Active indicates the question appears on the Questions panel.

•      Inactive indicates the question appeared on the Questions panel at one time, but no longer does.

Effective Date

Read only

The date the question first appears on the Questions panel

Question

Read only

The question text

Valid Answers

Read only

The possible answers to the question

Possible answers are Yes/No and sometimes N/A.

Explanation Required For

Read only

Indicates whether an explanation is required for the answer of Yes, No, or N/A

 

Date Required For

Read only

Indicates whether a review date is required for the answer of Yes, No, or N/A

Group Name

Read only

The panel where the question appears
