Parameters Tab: Panel Action Buttons

Figure 219 Parameters Tab with Action Buttons

 

Action Button

Description

Click to automatically enter information in the budget period fields on the Budget Periods and Totals panel. 

The data KC uses comes from data entered on the Personnel and Non-Personnel tabs for the first budget period. 

These calculations include data entered on the Rates tab.

Click only after making changes to rates on the Rates tab or after modifying budget details on line items that repeat from budget period to budget period.

Clicking Calculate All Periods applies your changes to all budget periods. 

Click to create successive budget periods in 365-day increments, beginning from the Proposal Start Date. 

All periods are one year long except for the last budget period, which may be shorter if the Proposal End Date is less than 1 year from the start date of the last budget period.

 

If you change the Proposal Start Date/ Proposal End Date on the Proposal Development document, click the default periods button to apply your date changes to the Parameters tab.
