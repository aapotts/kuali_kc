Award Transaction Type

The Award Transaction Type maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.

Edit Award Transaction Type Section

The Edit Award Transaction Type section of the Award Transaction Type maintenance document is a unique section that allows you to specify and maintain the details associated with the reference table record in fields.

Table 619 Award Transaction Type Maintenance Document - Edit Award Transaction Type Section Field Descriptions

Field

Description

Award Transaction Type Code

Required.  Enter a numeric value to uniquely identify this type of award transaction in KC.

Description

Required.  Enter the familiar, textual name of the type of award transaction.

Show In Action Summary

Required.  Click within the checkbox to select the item to show the award transaction type in the Action Summary.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.
