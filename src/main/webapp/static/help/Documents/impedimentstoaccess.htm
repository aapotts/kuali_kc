Impediments to Access

Document Locking:  In KC, locking rules are established to prevent data lost when a document is simultaneously accessed by more than one user. 

For more information about document locking, see “Document Locking” in E-Doc Fundamentals.

Browser Navigation Buttons:  As a general rule, using your Web browser’s navigation tools (for example, the Back button) while using the KC system is ill advised.  To maintain system integrity and avoid data loss, when navigating inside KC, use the navigation buttons and tools supplied as components of the KC user interface.

For more information about Web browser functionality as it relates to usage of the KC Web application, see “Browser Navigation Warning” in KC Overview.

System Timeout:  The timing out of your user session in the KC system also has an effect on your application access.

For more information about access duration limits, see “System Timeout” in Logging In.
