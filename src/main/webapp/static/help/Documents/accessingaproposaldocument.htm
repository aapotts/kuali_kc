•                                                         Create     Researcher Main Menu tab > Proposals menu group > Create Proposal (link)

•                                                         Open existing    Action List > Document Id column > (select document by ID #)

•                                                         Open existing    Doc Search > Search Results table > Document Id column > (link)

•         Access open proposal from Budget Document > return to proposal

For more information about accessing Proposal documents from Budget documents, see “Return To Proposal Navigation” in Budget Document.
