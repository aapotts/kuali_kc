Edit Award Type Panel

The Edit Award Type panel lets you create a new Award Type Code.  You must enter a code (abbreviation) for the new Award Type and a description for it.

 

Figure 477 Edit Award Type Panel

 

Table 179  Edit Award Type Panel Fields

Field

Definition

Award Type Code

Required

Enter a value to identify this Award Type.  The Award Type Code is numeric and must be unique.

Description

Required

Enter a descriptive name for this Award Type.
