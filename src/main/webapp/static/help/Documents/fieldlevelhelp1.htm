Field-Level Help

KC displays Field Level Help in a separate browser window.  It includes the field label, text describing the type of input accepted by the field, the purpose for the field, whether the field is required, and the maximum length of input allowed. 

Close the help screen by clicking the close button at the bottom of the pop-up window:
