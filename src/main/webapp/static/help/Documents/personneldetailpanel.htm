Personnel Detail Panel

Enter details for individual project personnel on this panel and its sub–panels. 

KC makes cost calculations based on the data entered for each project person listed and the rates entered on the Rates tab.  Rate Cost and Rate Cost Sharing figures show in a series of sub-panels sorted by Object Code Name.

 

Figure 244 Personnel Detail Panel Expanded

 

This panel has one main section: Add Details

At the top of the Add Details section, you can assign an Object Code Name to each project person. 

The Add Details section displays a sub-panel for each Object Code Name assigned to one or more personnel. 

Each of the Object Code Name sub-panels has two sub-panels of its own: Details and Rate Classes.
