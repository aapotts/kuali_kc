Valid Cost Element Job Code Maintenance Document

Use the Valid Cost Element Job Code maintenance document to establish new and to update Valid Cost Element Job Code combinations.

When your institution enters personnel budget line items on the Budget Document e-doc, you use valid Job Codes.

 

Figure 721 Budget Document Screen

 

On the Personnel tab of the Budget Document e-doc, you also use valid Cost Element/Job Code combinations.  When you add personnel to the budget, you must select a job code and an object code for each person.  After you select a Job Code, the valid Cost Element /Job Code combinations that your institution establishes determine which Object Codes KC displays for you on this screen:

When you create a new Valid Cost Element Job Code combination, you must enter or find and select both a Cost Element and a Job Code.  Follow your institution’s guidelines when creating a new valid Cost Element/Job Code combination.
