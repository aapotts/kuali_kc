IP Review Requirement Type

Use the Intellectual Property Review Requirement Type screen to look up, create, and update IP Review Req Type Codes.  IP Review Requirements specify the level of review for intellectual property items.  Expedited, Standard, and Special are examples of IP Review Requirement Types. IP Review Requirement appears on the Intellectual Property Review tab (Institutional Proposal).
